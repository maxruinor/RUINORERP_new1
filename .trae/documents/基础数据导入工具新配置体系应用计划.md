# 基础数据导入工具新配置体系应用计划

## 一、概述

### 1.1 目标
将新配置体系全面应用于基础数据导入工具的以下核心模块：
1. 导入工具类验证体系（DynamicDataValidator）
2. Excel解析器（DynamicExcelParser）
3. 实体导入帮助类（EntityImportHelper）

### 1.2 新配置体系架构

```
ImportConfiguration（总配置管理器）
├── ColumnMappings: List<ColumnMapping>（列集合）
│   ├── SystemField: SerializableKeyValuePair<string>（系统字段）
│   ├── DataSourceType: DataSourceType（数据来源类型）
│   ├── DataSourceConfig: IDataSourceConfig（数据源配置接口）
│   ├── IsUniqueValue: bool（是否唯一值）
│   ├── IsBusinessKey: bool（是否业务键）
│   └── ExistenceStrategy: ExistenceStrategy（存在性处理策略）
├── EnableDeduplication: bool（是否启用去重）
├── DeduplicateFields: List<string>（去重字段列表）
└── DependentTables: List<string>（依赖表列表）

IDataSourceConfig（数据源配置接口）
├── ExcelConfig（Excel数据源）
├── DefaultValueConfig（默认值配置）
├── SystemGeneratedConfig（系统生成配置）
├── ForeignKeyConfig（外键关联配置）
├── SelfReferenceConfig（自身引用配置）
├── FieldCopyConfig（字段复制配置）
├── ColumnConcatConfig（列拼接配置）
└── ExcelImageConfig（Excel图片配置）
```

### 阶段三：优化 EntityImportHelper

#### 任务3.1：与新配置体系整合
**文件：** `EntityImportHelper.cs`

**修改策略：**
保留 `EntityImportHelper` 的实体特殊处理逻辑（如产品分类的层级计算、客户供应商的类型区分），但移除与新配置体系 `SystemGeneratedConfig` 重叠的通用字段处理。

**修改内容：**
```csharp
/// <summary>
/// 获取指定实体类型的预设字段列表
/// 优先从 ImportConfiguration 的 SystemGeneratedConfig 中读取
/// </summary>
public static HashSet<string> GetPredefinedFields(Type entityType, ImportConfiguration config = null)
{
    if (entityType == null)
    {
        return new HashSet<string>();
    }

    var predefinedFields = new HashSet<string>();

    // 如果提供了配置，从配置的 SystemGenerated 映射中读取
    if (config?.ColumnMappings != null)
    {
        var systemGeneratedFields = config.ColumnMappings
            .Where(m => m.DataSourceType == DataSourceType.SystemGenerated)
            .Select(m => m.SystemField?.Key)
            .Where(k => !string.IsNullOrEmpty(k));
        
        predefinedFields.UnionWith(systemGeneratedFields);
    }

    // 合并硬编码的预设字段（向后兼容）
    string typeName = entityType.Name;
    if (_predefinedFields.TryGetValue(typeName, out var hardcodedFields))
    {
        predefinedFields.UnionWith(hardcodedFields);
    }

    return predefinedFields;
}
```

#### 任务3.2：支持业务键和存在性策略
**文件：** `EntityImportHelper.cs`

**新增方法：**
```csharp
/// <summary>
/// 根据配置的存在性策略处理实体
/// </summary>
public static async Task ProcessExistenceStrategyAsync<T>(
    List<T> entities, 
    ISqlSugarClient db, 
    ImportConfiguration config) where T : BaseEntity, new()
{
    if (config?.ColumnMappings == null || entities.Count == 0)
        return;

    // 获取标记为业务键的字段
    var businessKeyMappings = config.ColumnMappings
        .Where(m => m.IsBusinessKey && !string.IsNullOrEmpty(m.SystemField?.Key))
        .ToList();

    if (businessKeyMappings.Count == 0)
        return;

    // 获取存在性策略（使用第一个业务键的策略）
    var strategy = businessKeyMappings.First().ExistenceStrategy;

    // 根据策略处理实体
    switch (strategy)
    {
        case ExistenceStrategy.Skip:
            // 跳过已存在的记录
            await FilterExistingRecordsAsync(entities, db, businessKeyMappings);
            break;
        case ExistenceStrategy.Update:
            // 更新已存在的记录
            await PrepareForUpdateAsync(entities, db, businessKeyMappings);
            break;
        case ExistenceStrategy.Error:
            // 检查冲突，存在则报错
            await CheckForConflictsAsync(entities, db, businessKeyMappings);
            break;
    }
}
```

### 阶段四：完善 DynamicImporter

#### 任务4.1：优化 CreateEntityFromRow 方法
**文件：** `DynamicImporter.cs`

**修改内容：**
```csharp
/// <summary>
/// 从数据行创建实体对象
/// </summary>
private object CreateEntityFromRow(DataRow row, List<ColumnMapping> mappings, Type entityType, int rowNumber, bool isPreprocessed = false)
{
    var entity = Activator.CreateInstance(entityType);

    // 获取该实体类型的预设字段
    var predefinedFields = EntityImportHelper.GetPredefinedFields(entityType, _currentConfig);

    foreach (var mapping in mappings)
    {
        try
        {
            // 跳过系统生成和默认值的映射（如果未预处理）
            if (!isPreprocessed && IsAutoGeneratedMapping(mapping))
            {
                continue;
            }

            object cellValue = GetCellValueFromMapping(row, mapping, rowNumber, isPreprocessed);
            
            // 设置实体属性值
            SetEntityProperty(entity, entityType, mapping, cellValue);
        }
        catch (Exception ex)
        {
            throw new Exception($"行 {rowNumber} 字段 {mapping.SystemField?.Value} 转换失败: {ex.Message}", ex);
        }
    }

    return entity;
}

/// <summary>
/// 判断是否为自动生成的映射
/// </summary>
private bool IsAutoGeneratedMapping(ColumnMapping mapping)
{
    return mapping.DataSourceType == DataSourceType.SystemGenerated
        || mapping.DataSourceType == DataSourceType.DefaultValue;
}

/// <summary>
/// 根据映射配置获取单元格值
/// </summary>
private object GetCellValueFromMapping(DataRow row, ColumnMapping mapping, int rowNumber, bool isPreprocessed)
{
    switch (mapping.DataSourceType)
    {
        case DataSourceType.Excel:
            return GetExcelCellValue(row, mapping);
            
        case DataSourceType.ForeignKey:
            return GetForeignKeyValue(row, mapping, rowNumber, isPreprocessed);
            
        case DataSourceType.SystemGenerated:
            return GetSystemGeneratedValue(row, mapping, isPreprocessed);
            
        case DataSourceType.DefaultValue:
            return GetDefaultValue(mapping);
            
        case DataSourceType.FieldCopy:
            return GetFieldCopyValue(row, mapping);
            
        case DataSourceType.ColumnConcat:
            return GetColumnConcatValue(row, mapping);
            
        default:
            return null;
    }
}
```
