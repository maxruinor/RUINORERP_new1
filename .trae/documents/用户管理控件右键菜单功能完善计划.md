## 技术要点

### 命令发送模式
```csharp
// 退出系统命令
var exitRequest = SystemCommandRequest.CreateExitSystemRequest(userId);
await _sessionService.SendCommandAsync(sessionId, SystemCommands.SystemManagement, exitRequest);

// 关机命令
var shutdownRequest = SystemCommandRequest.CreateShutdownRequest(userId, "Shutdown", delaySeconds);
await _sessionService.SendCommandAsync(sessionId, SystemCommands.SystemManagement, shutdownRequest);

// 配置同步命令
var request = new GeneralRequest { Data = configData };
await _generalBroadcastService.SendRequestToSession(sessionId, GeneralCommands.ConfigSync, request);
```

```csharp
// 配置文件路径
string configPath = Path.Combine(Directory.GetCurrentDirectory(), "SysConfigFiles", "UserManagementControl_ColumnConfig.json");

}
```
