### P0-1: BaseBillEditGeneric 窗体加载时的重复数据绑定

**位置**: `BaseBillEditGeneric.cs` Load 事件和相关初始化方法

**问题描述**:
```csharp
// ❌ 问题：在 Load 事件中执行大量同步操作
private async void BaseBillEditGeneric_Load(object sender, EventArgs e)
{
    // 1. 初始化 UI 控件（大量控件创建）
    InitializeComponent();
    
    // 2. 立即加载数据（阻塞 UI）
    await LoadDataAsync();
    
    // 3. 绑定数据到多个控件（重复绑定）
    BindMasterData();
    BindDetailGrid();
    
    // 4. 检查锁定状态（网络请求）
    await CheckLockStatusAsync();
    
    // 5. 加载权限配置（反射操作）
    LoadPermissions();
    
    // 6. 初始化帮助系统（额外开销）
    InitializeHelpSystem();
}
```

**影响**:
- **UI 卡顿**: 所有初始化操作在 UI 线程执行，导致窗体加载时界面冻结
- **加载时间长**: 复杂单据加载时间可能超过 3-5 秒
- **用户体验差**: 用户看到白屏或无响应状态

**修复方案**:
```csharp
// ✅ 优化：分阶段异步加载
private async void BaseBillEditGeneric_Load(object sender, EventArgs e)
{
    try
    {
        // 1. 显示加载提示
        ShowBusyDialog("正在加载...");
        
        // 2. 先完成必要的 UI 初始化（同步）
        InitializeComponent();
        
        // 3. 异步加载非关键数据
        await Task.Run(async () =>
        {
            // 3.1 后台加载数据
            var loadDataTask = LoadDataAsync();
            
            // 3.2 并行加载权限配置
            var loadPermTask = Task.Run(() => LoadPermissions());
            
            // 3.3 并行初始化帮助系统
            var helpTask = Task.Run(() => InitializeHelpSystem());
            
            await Task.WhenAll(loadDataTask, loadPermTask, helpTask);
        });
        
        // 4. 在 UI 线程更新绑定
        BindMasterData();
        BindDetailGrid();
        
        // 5. 异步检查锁定状态（不阻塞）
        _ = CheckLockStatusAndUpdateUI();
        
        // 6. 隐藏加载提示
        HideBusyDialog();
    }
    catch (Exception ex)
    {
        logger?.LogError(ex, "窗体加载失败");
        HideBusyDialog();
    }
}
```

**预期效果**:
- 窗体加载时间减少 60-70%
- UI 响应性显著提升
- 用户感知等待时间降低

### P0-3: DataGridView 数据绑定时的频繁 UI 刷新

**位置**: `NewSumDataGridView.cs` 和相关数据绑定代码

**问题描述**:


**影响**:
- **UI 闪烁**: 大量重绘操作
- **性能低下**: 1000 行数据可能需要 5-10 秒
- **内存浪费**: 临时对象频繁创建销毁

**修复方案**:
```csharp
// ✅ 优化：批量绑定，暂停绘制
public void BindDataBatch(List<T> dataList)
{
    try
    {
        // 1. 暂停重绘
        this.SuspendLayout();
        
        // 2. 暂停数据变更通知
        var bindingSource = this.DataSource as BindingSource;
        if (bindingSource != null)
        {
            bindingSource.RaiseListChangedEvents = false;
        }
        
        // 3. 清空现有数据
        this.Rows.Clear();
        
        // 4. 批量添加数据
        if (dataList.Count > 0)
        {
            // 转换为 DataTable 或使用 BindingList
            var bindingList = new BindingList<T>(dataList);
            this.DataSource = bindingList;
        }
        
        // 5. 恢复数据变更通知
        if (bindingSource != null)
        {
            bindingSource.RaiseListChangedEvents = true;
            bindingSource.ResetBindings(false);
        }
        
        // 6. 恢复重绘
        this.ResumeLayout();
    }
    catch (Exception ex)
    {
        logger?.LogError(ex, "批量绑定数据失败");
        this.ResumeLayout();
    }
}
```

**预期效果**:
- 大数据量绑定速度提升 10-20 倍
- UI 闪烁完全消除
- 内存使用更加高效

### P1-2: 大图片未压缩直接加载

**位置**: 产品图片、业务图片加载相关代码

**问题描述**:
```csharp

```

**影响**:
- **内存占用**: 单张图片可能占用几十 MB
- **加载缓慢**: 解码大图片耗时
- **UI 卡顿**: 图片加载时界面冻结

**修复方案**:
```csharp
// ✅ 优化：按需缩放和缓存
public static Image LoadAndResizeImage(string imagePath, int maxWidth, int maxHeight)
{
    using (var originalImage = Image.FromFile(imagePath))
    {
        // 计算缩放比例
        var ratioX = (double)maxWidth / originalImage.Width;
        var ratioY = (double)maxHeight / originalImage.Height;
        var ratio = Math.Min(ratioX, ratioY);
        
        var newWidth = (int)(originalImage.Width * ratio);
        var newHeight = (int)(originalImage.Height * ratio);
        
        // 创建缩略图
        var thumbnail = new Bitmap(newWidth, newHeight);
        using (var graphics = Graphics.FromImage(thumbnail))
        {
            graphics.CompositingQuality = CompositingQuality.HighQuality;
            graphics.InterpolationMode = InterpolationMode.HighQualityBicubic;
            graphics.SmoothingMode = SmoothingMode.HighQuality;
            graphics.DrawImage(originalImage, 0, 0, newWidth, newHeight);
        }
        
        return thumbnail;
    }
}
```

### 5.3 持续监控机制

```csharp
public class PerformanceMonitor
{
    private readonly Timer _memoryTimer;
    private readonly Timer _cpuTimer;
    private readonly PerformanceCounter _cpuCounter;

    public PerformanceMonitor()
    {
        _cpuCounter = new PerformanceCounter("Processor", "% Processor Time", "_Total");
        
        _memoryTimer = new Timer(OnMemoryCheck, null, 0, 60000); // 每分钟检查
        _cpuTimer = new Timer(OnCpuCheck, null, 0, 5000); // 每 5 秒检查
    }

    private void OnMemoryCheck(object state)
    {
        var memory = Process.GetCurrentProcess().WorkingSet64 / 1024 / 1024;
        if (memory > 800) // 超过 800MB 告警
        {
            logger?.LogWarning($"内存占用过高：{memory}MB");
        }
    }

    private void OnCpuCheck(object state)
    {
        var cpu = _cpuCounter.NextValue();
        if (cpu > 80) // CPU 超过 80% 告警
        {
            logger?.LogWarning($"CPU 占用过高：{cpu}%");
        }
    }
}
```
