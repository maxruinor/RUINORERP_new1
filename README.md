# RUINORERP系统文档

## 概述

本文档旨在为RUINORERP企业级ERP系统的开发、维护和使用提供全面的技术参考资料。文档涵盖了系统架构概述和关键业务流程两个核心部分，帮助开发人员、系统管理员和业务用户快速了解系统的设计理念、技术实现和业务逻辑。

## 技术架构要点

RUINORERP采用客户端/服务器(C/S)架构，基于.NET Framework平台开发，主要技术组件包括：
- **通信框架**: SuperSocket
- **依赖注入**: Autofac
- **工作流引擎**: WorkflowCore
- **数据访问**: SqlSugar ORM
- **缓存管理**: CacheManager
- **界面框架**: WinForms
