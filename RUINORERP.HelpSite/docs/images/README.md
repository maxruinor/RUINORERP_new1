# 📸 截图添加指南

本指南说明如何为帮助文档添加界面截图。

## 📁 截图存放目录

```
docs/images/
├── UCSaleOrder/          # 销售订单截图目录
├── UCSaleOut/            # 销售出库截图目录
├── login/                # 登录界面截图目录
├── common/               # 通用截图目录
└── ...
```

## 🎯 截图命名规范

### 命名格式
```
功能-描述.png
```

### 命名示例

| 截图内容 | 文件名 |
|---------|--------|
| 销售订单主界面 | `main-interface.png` |
| 客户选择窗口 | `customer-select.png` |
| 创建订单操作 | `create-order.png` |
| 审核对话框 | `approve-dialog.png` |
| 打印预览 | `print-preview.png` |

### 命名规则

- ✅ 使用小写字母
- ✅ 使用连字符 `-` 分隔单词
- ✅ 使用有意义的描述
- ❌ 不使用中文
- ❌ 不使用空格
- ❌ 不使用特殊字符

## 📐 截图尺寸规范

### 推荐尺寸

| 用途 | 宽度 | 高度 | 格式 |
|------|------|------|------|
| 全界面截图 | 1280px 或 1920px | 自适应 | PNG |
| 局部截图 | 800px | 自适应 | PNG |
| 小图标/按钮 | 200px | 自适应 | PNG |

### 文件大小

- 单张截图不超过 **300KB**
- 如果太大，使用压缩工具优化

### 推荐工具

- **Windows**: Win + Shift + S (自带截图)
- **Snipaste**: 专业截图工具
- **ShareX**: 开源截图工具
- **Photoshop/GIMP**: 编辑和压缩

## 🔧 截图添加步骤

### 步骤1：创建目录

```bash
# 在销售订单目录下创建截图文件夹
mkdir docs/images/UCSaleOrder
```

### 步骤2：添加截图文件

将截图文件复制到对应目录：

```
docs/images/UCSaleOrder/
├── main-interface.png
├── customer-select.png
└── ...
```

### 步骤3：在文档中引用

在 `UCSaleOrder.md` 中添加：

```markdown
![销售订单界面](./images/UCSaleOrder/main-interface.png)
*销售订单主界面*
```

### 步骤4：验证显示

```bash
# 启动本地预览
mkdocs serve

# 访问 http://127.0.0.1:8000
# 检查截图是否正确显示
```

## 🎨 截图美化建议

### 标注工具

使用以下工具添加箭头、方框等标注：

- **Windows画图**：简单标注
- **Snipaste**：截图+标注
- **ShareX**：截图+标注+上传
- **FastStone Capture**：专业标注

### 标注规范

- 使用**红色**或**橙色**箭头指向关键位置
- 使用**黄色**方框圈出重要区域
- 添加简短文字说明
- 保持画面整洁，不要过度标注

### 示例

```
[截图示意]

┌─────────────────────────────────────┐
│  销售订单                           │
│                                     │
│  ┌─────────┐                       │
│  │客户 ▼   │ ← 点击选择客户       │
│  └─────────┘                       │
│                                     │
│  ┌─────────┐                       │
│  │保存     │ ← 点击保存订单       │
│  └─────────┘                       │
│                                     │
└─────────────────────────────────────┘
```

## ✅ 检查清单

添加截图后，请检查：

- [ ] 图片格式为 PNG
- [ ] 图片大小不超过 300KB
- [ ] 图片尺寸适中（1280px 或 1920px）
- [ ] 文件名符合命名规范
- [ ] 文件存放在正确目录
- [ ] Markdown引用路径正确
- [ ] 本地预览能正常显示
- [ ] 图片有必要的标注说明

## 📞 问题反馈

如果在添加截图时遇到问题：

1. 检查图片路径是否正确
2. 确认图片格式为 PNG 或 JPG
3. 查看浏览器控制台是否有404错误
4. 联系系统管理员寻求帮助
