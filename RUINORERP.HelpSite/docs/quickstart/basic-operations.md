# 基本操作

本文档介绍 RUINOR ERP 系统的常用基本操作。

## 📝 通用操作流程

### 新增记录

1. 点击工具栏的 **新建** 按钮（或按 `Ctrl + N`）
2. 填写表单信息
3. 点击 **保存** 按钮（或按 `Ctrl + S`）

![新增操作](../images/operation-add.png)

### 修改记录

**方法一**：在列表中双击要修改的记录

**方法二**：
1. 选中要修改的记录
2. 点击工具栏的 **修改** 按钮
3. 修改信息
4. 点击 **保存** 按钮

### 删除记录

1. 选中要删除的记录
2. 点击工具栏的 **删除** 按钮（或按 `Delete`）
3. 确认删除操作

!!! warning "注意"
    删除操作不可恢复，请谨慎操作！

### 查询记录

1. 点击工具栏的 **查询** 按钮（或按 `Ctrl + F`）
2. 在查询界面输入查询条件
3. 点击 **查询** 按钮
4. 查看查询结果

## 🔘 常用按钮说明

| 按钮 | 功能 | 使用场景 |
|------|------|---------|
| 💾 保存 | 保存当前编辑 | 填写完信息后保存 |
| 🔄 刷新 | 重新加载数据 | 数据可能已更新时 |
| 🖨️ 打印 | 打印当前单据 | 需要纸质单据时 |
| 📤 导出 | 导出为Excel/PDF | 需要数据分析时 |
| 📥 导入 | 从Excel导入数据 | 批量录入数据时 |
| ✓ 审核 | 审核当前单据 | 确认单据无误后 |
| ✗ 反审核 | 取消审核 | 需要修改已审核单据时 |

## 📋 表格操作

### 添加行

1. 点击表格下方的 **+** 按钮
2. 输入行数据
3. 按 `Enter` 或点击其他地方确认

### 删除行

1. 选中要删除的行
2. 点击 **-** 按钮
3. 确认删除

### 编辑行

1. 双击要编辑的单元格
2. 修改数据
3. 按 `Enter` 确认

### 排序

点击列标题即可按该列排序：

- 第一次点击：升序排列 ↑
- 第二次点击：降序排列 ↓
- 第三次点击：取消排序

## 🔍 高级查询技巧

### 模糊查询

使用 `%` 作为通配符：

```
查询"客户名称":
- 输入 "张%" → 查询所有姓张的客户
- 输入 "%公司%" → 查询名称中包含"公司"的客户
- 输入 "北京%" → 查询所有以"北京"开头的客户
```

### 日期范围查询

```
查询日期范围：
- 开始日期：2024-01-01
- 结束日期：2024-01-31
→ 查询1月份的所有记录
```

### 多条件组合

可以同时设置多个查询条件，条件之间是**并且**关系：

```
示例：
- 客户：张三
- 日期：2024-01-01 至 2024-01-31
- 金额：> 1000
→ 查询张三在1月份金额大于1000的订单
```

## 💡 操作技巧

### 快捷键

掌握快捷键可以大幅提升效率：

| 操作 | 快捷键 |
|------|--------|
| 新建 | `Ctrl + N` |
| 保存 | `Ctrl + S` |
| 删除 | `Delete` |
| 查询 | `Ctrl + F` |
| 打印 | `Ctrl + P` |
| 帮助 | `F1` |

完整快捷键列表请查看 [快捷键指南](shortcuts.md)。

### 批量操作

在列表界面可以批量选择记录：

- `Ctrl + 点击` - 多选不连续记录
- `Shift + 点击` - 选择连续范围
- `Ctrl + A` - 全选

### 数据验证

系统会在保存时自动验证数据：

- ✅ **必填项**：标有红色星号 * 的字段必须填写
- ✅ **格式校验**：日期、金额等会自动校验格式
- ✅ **业务规则**：如信用额度、库存数量等会自动检查
