## 💡 使用技巧

### F1 帮助系统

在系统的任何界面，按下 `F1` 键即可打开当前功能的帮助文档：

- 在**销售订单**界面按 `F1` → 打开销售订单帮助
- 在**客户**字段按 `F1` → 打开客户字段帮助
- 在**查询界面**按 `F1` → 打开查询功能帮助

### 实时更新

本帮助文档支持实时更新：

- ✅ 在线时自动获取最新内容
- ✅ 离线时显示本地缓存版本
- ✅ 网络恢复后自动同步

### 搜索功能

使用页面右上角的搜索框，可以快速找到相关内容：

```
搜索示例：
- 输入"销售订单" → 找到销售订单相关帮助
- 输入"客户" → 找到客户管理相关帮助
- 输入"F1" → 找到帮助系统使用说明
```

## 📚 模块速查

| 模块 | 主要功能 | 快捷键 |
|------|---------|--------|
| 销售管理 | 销售订单、出库、退货 | `Alt + S` |
| 采购管理 | 采购订单、入库、退货 | `Alt + P` |

## ❓ 常见问题

**Q: 为什么按 F1 打不开帮助？**  
A: 请检查网络连接，系统会自动切换到本地帮助。

**Q: 帮助内容显示不全？**  
A: 尝试刷新页面或检查网络连接。

**Q: 如何查看历史版本的帮助？**  
A: 点击页面底部的"最后更新"链接查看更新历史。
