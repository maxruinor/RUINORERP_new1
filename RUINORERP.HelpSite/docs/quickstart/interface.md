## 📋 功能区域说明

### 1. 顶部菜单栏

包含系统的所有功能菜单：

- **文件(F)** - 新建、打开、保存、打印、退出
- **编辑(E)** - 撤销、重做、复制、粘贴
- **视图(V)** - 界面布局、工具栏显示/隐藏
- **工具(T)** - 计算器、日历、系统设置
- **窗口(W)** - 窗口排列、切换
- **帮助(H)** - 帮助文档、关于系统

### 2. 工具栏

常用功能的快捷按钮：

| 图标 | 功能 | 快捷键 |
|------|------|--------|
| 🏠 | 返回首页 | `Alt + Home` |
| 📁 | 打开 | `Ctrl + O` |
| 💾 | 保存 | `Ctrl + S` |
| 🖨️ | 打印 | `Ctrl + P` |
| 🔍 | 查找 | `Ctrl + F` |
| ❓ | 帮助 | `F1` |

### 3. 左侧导航栏

业务模块快捷入口：

- 📦 **销售管理**
  - 销售订单
  - 销售出库
  - 销售退货
  - 客户管理
  
- 🛒 **采购管理**
  - 采购订单
  - 采购入库
  - 采购退货
  - 供应商管理
  
- 📦 **库存管理**
  - 库存查询
  - 库存盘点
  - 库存调拨
  
- 💰 **财务管理**
  - 应收管理
  - 应付管理
  - 收款单
  - 付款单
  
- ⚙️ **系统设置**
  - 用户管理
  - 权限设置
  - 参数配置

### 4. 工作区

主要操作区域，显示当前打开的窗体：

- 支持多标签页
- 支持拖拽调整大小
- 支持分屏显示

### 5. 状态栏

显示系统状态信息：

- **就绪** - 当前操作状态
- **用户** - 当前登录用户
- **服务器** - 连接状态（在线/离线）
- **时间** - 当前系统时间

## 🎨 界面自定义

### 调整布局

1. **拖动边框** - 调整各区域大小
2. **右键菜单** - 显示/隐藏工具栏
3. **视图菜单** - 切换界面主题

### 保存布局

系统会自动保存您的界面布局，下次登录时恢复。
