# 系统登录

本文档介绍如何登录 RUINOR ERP 系统。

## 🔐 登录步骤

### 1. 启动程序

双击桌面上的 **RUINOR ERP** 图标，启动客户端程序。

![登录界面](../images/login-screen.png)
*登录界面示意图（请添加实际截图）*

### 2. 输入服务器地址

在**服务器地址**栏输入服务器 IP 或域名：

```
格式：IP:端口 或 域名:端口
示例：
- 192.168.1.100:8888
- erp.company.com:8888
```

!!! tip "提示"
    如果不知道服务器地址，请联系您的系统管理员。

### 3. 输入用户名和密码

- **用户名**：您的员工编号或系统账号
- **密码**：初始密码通常为 `123456` 或管理员分配的密码

!!! warning "安全提示"
    首次登录后请立即修改密码，建议使用字母+数字+符号的组合。

### 4. 点击登录

点击**登录**按钮，等待系统验证。

## ✅ 登录成功

登录成功后，系统将显示主界面：

### 主界面组成

1. **顶部菜单栏** - 系统功能菜单
2. **左侧导航栏** - 业务模块快捷入口
3. **中间工作区** - 当前操作界面
4. **底部状态栏** - 显示连接状态、用户信息等

## ❌ 登录失败

### 常见错误及解决方案

| 错误提示 | 原因 | 解决方案 |
|---------|------|---------|
| "无法连接到服务器" | 网络不通或服务器未启动 | 检查网络连接，联系管理员 |
| "用户名或密码错误" | 账号信息有误 | 检查大小写，确认账号状态 |
| "用户已被锁定" | 多次输错密码 | 联系管理员解锁账号 |
| "版本不匹配" | 客户端版本过旧 | 运行自动更新或重新安装 |

### 忘记密码

如果忘记密码，请联系系统管理员重置。
