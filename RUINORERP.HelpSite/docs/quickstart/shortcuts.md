# 快捷键指南

掌握快捷键可以大幅提升您的工作效率！

## ⌨️ 通用快捷键

### 文件操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl + N` | 新建 | 新建一条记录 |
| `Ctrl + O` | 打开 | 打开查询界面 |
| `Ctrl + S` | 保存 | 保存当前记录 |
| `Ctrl + P` | 打印 | 打印当前单据 |
| `Ctrl + F` | 查找 | 打开查询界面 |

### 编辑操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl + Z` | 撤销 | 撤销上一步操作 |
| `Ctrl + Y` | 重做 | 重做撤销的操作 |
| `Ctrl + C` | 复制 | 复制选中的内容 |
| `Ctrl + V` | 粘贴 | 粘贴复制的内容 |
| `Delete` | 删除 | 删除选中的记录 |

### 系统操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `F1` | 帮助 | 打开当前界面的帮助文档 |
| `F5` | 刷新 | 刷新当前界面数据 |
| `F11` | 全屏 | 切换全屏模式 |
| `Alt + F4` | 退出 | 退出系统 |

## 🔘 按钮快捷键

界面上带下划线的字母表示快捷键：

```
示例：
[保存(S)] → 按 Alt + S 触发
[审核(A)] → 按 Alt + A 触发
[关闭(C)] → 按 Alt + C 触发
```

## 💡 效率技巧

### 快速导航

- `Alt` 键显示所有可用快捷键
- 连续按快捷键组合快速切换功能

### 批量操作

```
批量选择：
- Ctrl + A        全选
- Ctrl + 点击     多选不连续项
- Shift + 点击    选择连续范围
- Ctrl + 空格     切换选择状态
```

### 表单快速填写

| 快捷键 | 功能 |
|--------|------|
| `Tab` | 下一个字段 |
| `Shift + Tab` | 上一个字段 |
| `Enter` | 确认并下一个 |
| `Esc` | 取消编辑 |
| `F2` | 进入编辑模式 |

## 🎓 学习建议

### 初学者路径

1. **第一周**：掌握 `Ctrl + N/S/F/P` 和 `F1`
2. **第二周**：学习模块快捷键 `Alt + ?`
3. **第三周**：熟练表格操作快捷键
4. **第四周**：掌握所有快捷键，实现盲操作

### 每日练习

```
建议：
- 每天使用快捷键代替鼠标操作
- 记录常用的操作，针对性练习
- 坚持使用，21天养成习惯
```

### 效率对比

| 操作 | 鼠标点击 | 快捷键 | 节省时间 |
|------|---------|--------|---------|
| 新建 + 保存 | 4次点击 | Ctrl+N, Ctrl+S | 70% |
| 打开查询 | 3次点击 | Ctrl+F | 60% |
| 打印单据 | 3次点击 | Ctrl+P | 60% |
| 打开帮助 | 2次点击 | F1 | 80% |

**平均可提升 50-70% 的操作效率！**
