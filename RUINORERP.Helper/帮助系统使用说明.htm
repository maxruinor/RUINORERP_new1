RUINOR ERP 帮助系统使用说明

1. 系统登录和基本操作

1.1 首次登录配置

步骤1：服务器配置

首次使用系统时，需要配置服务器连接：

  1. 打开系统登录界面
  2. 点击右上角的"服务器设置"按钮
  3. 输入服务器IP地址和端口号（默认端口：8080）
  4. 点击"测试连接"验证配置是否正确
  5. 保存配置并重新启动系统
步骤2：用户登录
  1. 输入用户名和密码
  2. 选择登录的账套（如果有多账套）
  3. 点击"登录"按钮进入系统
提示：如果忘记密码，请联系系统管理员重置密码。

1.2 主界面导航

登录成功后，系统显示主工作台界面，包含以下主要区域：

  • 顶部菜单栏：系统功能模块导航
  • 左侧功能树：具体功能菜单
  • 工作区：数据展示和操作区域
  • 状态栏：显示系统状态和用户信息

2. 核心业务流程操作指南

2.1 销售订单处理流程

创建销售订单
  1. 点击菜单：销售管理 → 销售订单 → 新建订单
  2. 选择客户：点击客户字段旁的搜索按钮选择客户
  3. 添加商品：点击"添加商品"按钮，选择商品和数量
  4. 设置价格：输入单价，系统自动计算总金额
  5. 保存订单：点击按钮
订单审核流程
  1. 订单保存后，点击按钮
  2. 系统根据订单金额自动分配审核人员
  3. 审核人员在工作台看到待审核订单
  4. 审核通过后，订单状态变为"已审核"
  5. 仓库人员可以查看已审核订单准备出库
注意：金额超过500元的订单需要指定人员审核，500元以下订单自动审核通过。

2.2 采购订单处理流程

创建采购申请
  1. 点击菜单：采购管理 → 采购申请 → 新建申请
  2. 填写申请部门、申请原因、所需物料
  3. 提交申请给部门主管审批
生成采购订单
  1. 采购人员在"待处理申请"中查看已批准的申请
  2. 点击"生成采购订单"按钮
  3. 选择供应商、确定采购价格和交货日期
  4. 保存并提交采购订单

2.3 财务收款流程

重要：财务收款分为预收款和应收款两种处理方式。
预收款处理（销售出库前）
  1. 客户预付货款时，点击：财务管理 → 预收款 → 新建预收款
  2. 选择对应的销售订单
  3. 输入收款金额和收款方式
  4. 系统生成预收款凭证
应收款处理（销售出库后）
  1. 销售出库后，系统自动生成应收款
  2. 财务人员查看：财务管理 → 应收款管理
  3. 如果有预收款，系统自动核销
  4. 没有预收款时，手动创建收款单
收款单处理方式
  1. 方式一：在应收款界面点击"转收款单"
  2. 方式二：通过对账功能生成收款单
  3. 输入实际收款信息，完成收款确认

3. 退货退款操作指南

3.1 销售退货流程

创建退货申请
  1. 点击菜单：销售管理 → 销售退货 → 新建退货
  2. 选择原销售订单
  3. 填写退货原因和退货数量
  4. 提交退货申请
退货审批和处理
  1. 销售部门审批退货申请
  2. 质量部门确认退货原因
  3. 仓库接收退货商品
  4. 财务部门处理退款

3.2 采购退货流程

创建采购退货
  1. 点击菜单：采购管理 → 采购退货 → 新建退货
  2. 选择原采购订单
  3. 填写退货原因和数量
  4. 提交给供应商确认

4. 数据录入注意事项

4.1 必填字段识别

在数据录入界面，带红色星号(*)的字段为必填项：

  • 客户名称、商品名称、数量、单价等
  • 日期字段通常有默认值，但需要确认
  • 金额字段系统会自动计算，但需要核对

4.2 数据验证规则

  • 数量：必须为大于0的数字
  • 单价：支持小数点后2位
  • 日期：不能早于系统启用日期
  • 客户/供应商：必须从列表中选择

4.3 快捷操作技巧

  • F1键：在任何界面按F1查看帮助
  • Ctrl+S：快速保存当前数据
  • Ctrl+F：在当前界面快速查找
  • F5键：刷新数据列表
  • Tab键：在字段间快速切换

5. 常见问题解答

Q: 如何修改已提交的订单？

A: 已提交的订单需要先点击"反审核"按钮，修改后再重新提交审核。

Q: 订单审核需要多长时间？

A: 系统会根据订单金额自动分配审核人员，一般会在24小时内完成审核。

Q: 预收款和应收款有什么区别？

A: 预收款是销售出库前的收款，应收款是出库后生成的债权。系统会自动处理两者的核销关系。

Q: 如何查看订单的处理状态？

A: 在订单查询界面，可以通过状态筛选查看订单的当前处理进度。

Q: 忘记密码怎么办？

A: 请联系系统管理员重置密码，重置后首次登录需要修改密码。
