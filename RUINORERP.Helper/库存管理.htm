库存管理

库存管理模块用于管理企业的仓库和库存业务，包括入库、出库、调拨、盘点等操作。

功能概述

库存管理主要包括仓库档案管理、库存流水管理、库存调拨、库存盘点、库存分析等业务。

仓库管理

  • 维护仓库基本信息
  • 管理库位和货架信息
  • 设置仓库权限和责任人
  • 定义仓库作业流程

库存流水管理

  • 记录所有出入库操作
  • 实时更新库存余额
  • 提供库存流水查询
  • 支持批次和序列号管理

库存调拨管理

  • 处理不同仓库间的货品调拨
  • 记录调拨申请和审批
  • 跟踪调拨执行状态
  • 更新调出和调入仓库库存

操作流程

库存盘点流程

  1. 制定盘点计划
  2. 冻结盘点仓库业务
  3. 实地盘点并录入盘点结果
  4. 审核盘点数据
  5. 生成盘盈盘亏单据
  6. 处理盘盈盘亏并调整库存

库存盘点审核流程

库存盘点完成后需要经过审核流程才能生效：

  1. 盘点数据录入完成后，状态为"未审核"
  2. 具有审核权限的用户可以对盘点数据进行审核操作
  3. 审核时需要填写审核意见，选择"同意"或"拒绝"
  4. 审核同意后，盘点数据状态更新为"已审核"，系统自动生成盘盈盘亏单据
  5. 审核拒绝后，盘点数据状态保持"未审核"，需要修改后重新提交审核

库存盘点反审核流程

对于已审核但尚未结案的盘点数据，可以执行反审核操作：

  1. 选择需要反审核的已审核盘点数据
  2. 点击"反审核"按钮，系统会提示确认操作
  3. 确认后，盘点数据状态从"已审核"回退到"未审核"状态
  4. 反审核后的盘点数据可以进行修改操作，修改后需要重新提交审核

库存盘点结案流程

库存盘点在完成所有盘盈盘亏处理后可以进行结案：

  1. 确保盘点数据已审核且所有盘盈盘亏已处理完成
  2. 选择需要结案的盘点数据
  3. 点击"结案"按钮，系统会提示确认操作
  4. 确认后，盘点数据状态更新为"已结案"，不能再进行修改操作

库存调拨流程

  1. 创建调拨申请单
  2. 填写调拨明细信息
  3. 提交调拨申请审批
  4. 审批通过后执行调拨
  5. 调出仓库发货确认
  6. 调入仓库收货确认

库存调拨审核流程

库存调拨申请创建后需要经过审核流程才能生效：

  1. 调拨申请创建完成后，状态为"未审核"
  2. 具有审核权限的用户可以对调拨申请进行审核操作
  3. 审核时需要填写审核意见，选择"同意"或"拒绝"
  4. 审核同意后，调拨申请状态更新为"已审核"，可以进行后续调拨操作
  5. 审核拒绝后，调拨申请状态保持"未审核"，需要修改后重新提交审核

库存调拨反审核流程

对于已审核但尚未结案的调拨申请，可以执行反审核操作：

  1. 选择需要反审核的已审核调拨申请
  2. 点击"反审核"按钮，系统会提示确认操作
  3. 确认后，调拨申请状态从"已审核"回退到"未审核"状态
  4. 反审核后的调拨申请可以进行修改操作，修改后需要重新提交审核

库存调拨结案流程

库存调拨在完成所有调出调入操作后可以进行结案：

  1. 确保调拨申请已审核且所有货物已完成调拨
  2. 选择需要结案的调拨申请
  3. 点击"结案"按钮，系统会提示确认操作
  4. 确认后，调拨申请状态更新为"已结案"，不能再进行修改操作

关键字段说明

字段名 说明
仓库 货物存放的仓库名称
库位 货物存放的具体位置
实际库存 当前仓库中的实际货物数量
在途库存 正在运输中的货物数量
可用库存 可以使用的货物数量
安全库存 为保证正常运营所需的最低库存量
最高库存 仓库可容纳的最大库存量
库存成本 当前库存货物的总成本
