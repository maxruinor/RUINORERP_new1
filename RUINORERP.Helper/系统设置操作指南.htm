系统设置操作指南

1. 系统设置概述

1.1 系统设置功能

系统设置模块用于配置RUINOR ERP系统的各项参数，确保系统正常运行并满足企业的个性化需求。只有具有系统管理员权限的用户才能进行系统设置操作。

1.2 设置分类

设置类别主要功能操作权限
基础参数公司信息、系统参数系统管理员
用户权限用户管理、角色权限系统管理员
业务参数业务流程、审批设置相关部门主管
数据字典基础数据、编码规则数据管理员

2. 服务器配置

2.1 首次服务器配置

配置步骤：
  1. 打开系统登录界面
  2. 点击右上角的按钮
  3. 输入服务器信息：
    • 服务器地址：ERP服务器IP地址
    • 端口号：默认8080（根据实际设置）
    • 数据库：选择连接的数据库
  4. 点击验证配置
  5. 保存配置并重新启动系统
提示：服务器配置信息由系统实施人员提供，如有疑问请联系技术支持。

2.2 多服务器配置

配置说明：
  1. 系统支持连接多个服务器实例
  2. 可以设置主服务器和备用服务器
  3. 主服务器故障时自动切换到备用服务器
  4. 确保业务连续性和数据安全

3. 用户权限管理

3.1 用户账号管理

新建用户：
  1. 点击菜单：系统设置 → 用户管理
  2. 点击按钮
  3. 填写用户基本信息：
    • 用户名：登录用户名（英文或数字）
    • 真实姓名：用户中文姓名
    • 部门：选择所属部门
    • 职位：用户职位信息
    • 邮箱：用于接收系统通知
  4. 设置初始密码
  5. 分配用户角色和权限

3.2 角色权限设置

权限配置：
  1. 点击菜单：系统设置 → 角色管理
  2. 选择要配置的角色（如：销售员、采购员、财务等）
  3. 设置功能权限：勾选该角色可以访问的功能模块
  4. 设置数据权限：定义该角色可以查看和操作的数据范围
  5. 保存权限设置

3.3 密码策略设置

安全策略：
  1. 密码复杂度：设置密码最小长度和复杂度要求
  2. 密码有效期：设置密码需要定期更换
  3. 登录失败锁定：连续登录失败自动锁定账户
  4. 会话超时：设置用户会话自动超时时间

4. 基础数据设置

4.1 公司信息设置

公司资料：
  1. 点击菜单：系统设置 → 公司信息
  2. 填写公司基本信息：
    • 公司名称
    • 公司地址
    • 联系电话
    • 税务登记号
    • 银行账户信息
  3. 上传公司Logo
  4. 保存设置

4.2 数据字典管理

字典维护：
  1. 点击菜单：系统设置 → 数据字典
  2. 选择字典分类（如：性别、职务、产品类别等）
  3. 可以查看现有字典项列表
  4. 点击添加新的字典项
  5. 设置字典项的编码、名称、排序等

5. 业务流程设置

5.1 审批流程设置

流程配置：
  1. 点击菜单：系统设置 → 审批流程
  2. 选择业务类型（如：销售订单、采购订单、费用报销等）
  3. 设置审批节点和审批人员
  4. 配置审批条件（如：金额阈值、部门等）
  5. 设置审批时限和提醒方式

5.2 业务参数设置

参数配置：
  1. 财务参数：默认币种、小数位数、税率等
  2. 库存参数：安全库存天数、盘点周期等
  3. 销售参数：默认价格策略、折扣规则等
  4. 采购参数：供应商评估标准、采购周期等

6. 系统维护

6.1 数据备份

备份操作：
  1. 点击菜单：系统设置 → 数据备份
  2. 设置备份类型：完整备份、增量备份
  3. 选择备份路径
  4. 点击
  5. 系统显示备份进度和结果

6.2 数据恢复

恢复操作：
  1. 点击菜单：系统设置 → 数据恢复
  2. 选择要恢复的备份文件
  3. 确认恢复操作（此操作会覆盖现有数据）
  4. 系统执行数据恢复
  5. 恢复完成后重新启动系统
重要：数据恢复操作会覆盖现有数据，请谨慎操作并确保有最新备份。

6.3 系统日志查看

日志管理：
  1. 点击菜单：系统设置 → 系统日志
  2. 可以查看各种类型的日志：
    • 操作日志：用户操作记录
    • 安全日志：登录和权限变更记录
    • 系统日志：系统运行状态记录
    • 错误日志：系统错误和异常记录
  3. 支持按时间、用户、模块等条件查询
  4. 可以导出日志文件进行分析

7. 个性化设置

7.1 界面个性化

界面设置：
  1. 主题设置：选择界面颜色主题
  2. 语言设置：选择系统显示语言
  3. 布局设置：调整界面布局和字体大小
  4. 快捷方式：设置常用功能的快捷方式

7.2 个人设置

个人偏好：
  1. 点击右上角用户头像 → 个人设置
  2. 修改密码：定期更改登录密码
  3. 通知设置：设置接收哪些系统通知
  4. 工作台设置：自定义工作台显示内容

8. 常见问题解答

Q: 如何重置用户密码？

A: 在用户管理界面选择要重置密码的用户，点击"重置密码"按钮，系统会生成临时密码发送到用户邮箱。

Q: 新增的功能模块如何分配权限？

A: 在角色管理界面，为新模块勾选相应的权限，然后分配给相应用户角色。

Q: 数据备份的频率应该是多少？

A: 建议每天进行增量备份，每周进行完整备份，重要数据变更后立即备份。

Q: 系统日志文件太大如何处理？

A: 可以设置日志自动清理策略，定期清理过期日志文件。

Q: 如何查看系统版本信息？

A: 在帮助菜单中点击"关于系统"可以查看版本信息和版权信息。

9. 操作注意事项

权限控制：系统设置涉及系统安全和稳定性，必须由授权人员操作。
备份优先：在进行重要设置变更前，务必先进行数据备份。
测试验证：重要设置变更后，需要在测试环境验证后再应用到生产环境。
文档记录：重要的系统设置变更应该记录在变更日志中。
