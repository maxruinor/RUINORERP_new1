财务收款操作指南

1. 收款流程概述

1.1 两种收款方式

RUINOR ERP系统支持两种收款处理方式：

预收款（销售出库前）
  • 客户在下单后、货物出库前预付货款
  • 适用于需要预付款的订单
  • 系统生成预收款凭证，关联销售订单
应收款转收款（销售出库后）
  • 货物出库后系统自动生成应收款
  • 财务人员根据实际收款情况处理
  • 系统支持自动核销和手动收款
重要概念：预收款是负债（客户预付款），应收款是资产（客户欠款）。

2. 预收款处理

2.1 创建预收款单

操作路径：
  1. 点击菜单：财务管理 → 预收款管理
  2. 点击按钮
  3. 选择对应的销售订单
  4. 系统自动带出客户信息和订单金额

2.2 填写预收款信息

必填信息：
  1. 收款日期：实际收款日期
  2. 收款金额：实际收到的预付款金额
  3. 收款方式：现金、转账、支票等
  4. 银行账户：收款入账的银行账户
  5. 备注信息：收款相关说明

2.3 保存预收款单

保存操作：
  1. 确认信息无误后，点击按钮
  2. 系统生成预收款凭证编号（如：PR202501030001）
  3. 预收款单状态变为"已确认"
  4. 系统自动更新销售订单的预收款信息
提示：预收款单保存后，对应的销售订单会显示已收预付款金额。

3. 应收款管理

3.1 应收款生成

自动生成流程：
  1. 销售订单审核通过后，仓库进行出库操作
  2. 出库单确认后，系统自动生成应收款
  3. 应收款金额 = 订单总额 - 已收预付款
  4. 财务人员可以在应收款管理界面查看

3.2 应收款查询

查询路径：
  1. 点击菜单：财务管理 → 应收款管理
  2. 设置查询条件：客户、日期范围、状态等
  3. 点击按钮
  4. 系统显示应收款列表，包含以下信息：
    • 应收款编号
    • 客户名称
    • 应收金额
    • 已收金额
    • 欠款金额
    • 到期日期
    • 账龄分析

4. 收款单处理

4.1 方式一：应收款转收款单

操作步骤：
  1. 在应收款管理界面，选择需要收款的记录
  2. 点击按钮
  3. 系统自动创建收款单，带出应收款信息
  4. 填写实际收款信息：
    • 收款日期
    • 实收金额（可部分收款）
    • 收款方式
    • 银行账户
  5. 点击完成收款

4.2 方式二：对账生成收款单

对账操作：
  1. 点击菜单：财务管理 → 对账管理
  2. 选择客户，设置对账期间
  3. 系统显示该期间内的所有交易记录
  4. 核对确认后，点击
  5. 系统根据对账结果创建收款单

4.3 收款单核销

核销规则：
  1. 有预收款的情况：系统自动核销预收款
  2. 无预收款的情况：收款单直接冲销应收款
  3. 部分收款：按收款金额比例核销
  4. 多笔订单：支持合并收款和分账核销

5. 自动核销流程

5.1 预收款自动核销

核销时机：
  1. 销售订单出库生成应收款时
  2. 系统自动检查该订单是否有预收款
  3. 如有预收款，自动进行核销处理
  4. 核销后，应收款金额相应减少

5.2 核销结果查看

查看路径：
  1. 在应收款详情界面，查看"已核销金额"
  2. 在预收款详情界面，查看"已核销状态"
  3. 在收款单界面，查看核销明细记录

6. 收款状态管理

6.1 收款状态说明

状态说明后续操作
待收款应收款已生成，等待收款创建收款单
部分收款已收到部分款项继续收款或核销
已收款款项已全部收回查看收款记录
逾期未收超过付款期限未收款催款处理

6.2 逾期账款处理

催款流程：
  1. 系统自动标记逾期应收款
  2. 财务人员查看逾期账款列表
  3. 生成催款通知单
  4. 记录催款过程和结果
  5. 更新账款状态

7. 财务报表查询

7.1 收款统计报表

报表路径：
  1. 点击菜单：财务管理 → 报表查询
  2. 选择：收款统计报表
  3. 设置统计期间和统计维度
  4. 系统生成收款分析报表，包含：
    • 期间收款总额
    • 各客户收款情况
    • 收款方式分布
    • 逾期账款分析

7.2 账龄分析报表

账龄分析：
  1. 查看各时间段内的应收账款
  2. 分析账款回收情况
  3. 识别风险客户和账款
  4. 为催款和坏账准备提供依据

8. 常见问题解答

Q: 预收款和收款单有什么区别？

A: 预收款是销售前的收款，属于负债；收款单是销售后的收款，用于冲销应收款。

Q: 如何修改已保存的收款单？

A: 已确认的收款单不能直接修改，需要先作废原收款单，然后重新创建。

Q: 部分收款如何核销多笔订单？

A: 系统支持按比例核销，也可以手动指定每笔订单的核销金额。

Q: 收款单作废后会影响什么？

A: 作废收款单会取消对应的核销记录，应收款状态会相应调整。

Q: 如何查看收款历史记录？

A: 在客户详情界面或收款查询界面，可以查看该客户的所有收款记录。

9. 操作注意事项

重要提醒：收款操作涉及资金流动，务必仔细核对金额和账户信息。
数据一致性：确保收款信息与银行流水一致，避免账务差异。
及时性：收到款项后应及时录入系统，确保财务数据的实时性。
