采购管理操作指南

1. 采购业务流程概述

1.1 完整采购流程

标准采购流程：
  1. 采购申请 → 部门审批
  2. 采购订单 → 供应商确认
  3. 货物入库 → 质量检验
  4. 发票核对 → 财务付款

1.2 采购相关角色

角色职责主要操作
申请人员提出采购需求创建采购申请
部门主管审批采购申请审核采购需求
采购人员执行采购创建采购订单
仓库人员接收货物入库验收
财务人员付款处理核对发票付款

2. 采购申请管理

2.1 创建采购申请

操作路径：
  1. 点击菜单：采购管理 → 采购申请
  2. 点击按钮
  3. 填写申请基本信息：
    • 申请部门（自动带出）
    • 申请人（自动带出）
    • 申请日期（系统默认）
    • 需求日期（预计需要时间）
    • 申请原因（必填）

2.2 添加申请明细

明细添加：
  1. 点击按钮
  2. 选择需要的物料或商品
  3. 输入申请数量
  4. 填写用途说明
  5. 系统自动计算预估金额

2.3 提交审批

审批流程：
  1. 填写完成后点击
  2. 系统根据金额自动分配审批人员
  3. 审批人员在工作台看到待审批申请
  4. 审批通过后，采购人员可以看到已批准的申请

3. 采购订单管理

3.1 生成采购订单

从申请生成订单：
  1. 采购人员进入：采购管理 → 待处理申请
  2. 选择已批准的采购申请
  3. 点击
  4. 系统自动创建采购订单，带出申请信息

3.2 直接创建采购订单

直接创建：
  1. 点击菜单：采购管理 → 采购订单
  2. 点击
  3. 选择供应商（必填）
  4. 添加采购商品和数量
  5. 设置交货日期和价格条款

3.3 采购订单审核

审核流程：
  1. 订单保存后点击
  2. 系统根据金额判断审核流程
  3. 审核通过后订单状态变为"已审核"
  4. 供应商可以查看已审核的订单

4. 采购入库管理

4.1 创建入库单

入库操作：
  1. 仓库人员进入：库存管理 → 采购入库
  2. 选择已审核的采购订单
  3. 点击
  4. 系统自动创建入库单，带出订单信息

4.2 入库验收

验收流程：
  1. 核对到货数量与订单数量
  2. 进行质量检验
  3. 填写实际入库数量
  4. 如有差异，填写差异原因
  5. 确认入库，更新库存

4.3 入库状态管理

状态说明后续操作
待入库订单已审核，等待到货生成入库单
部分入库部分货物已入库继续入库
已入库全部货物入库完成生成应付款
入库异常到货数量或质量有问题联系供应商处理

5. 采购退货管理

5.1 创建退货单

退货流程：
  1. 点击菜单：采购管理 → 采购退货
  2. 选择需要退货的采购订单
  3. 点击
  4. 填写退货原因和退货数量
  5. 提交退货申请

5.2 退货审批

审批流程：
  1. 采购部门审批退货申请
  2. 质量部门确认退货原因
  3. 供应商确认接收退货
  4. 仓库进行退货出库操作

6. 供应商管理

6.1 供应商信息维护

供应商资料：
  1. 点击菜单：基础资料 → 供应商管理
  2. 可以查看现有供应商列表
  3. 点击添加新供应商
  4. 填写供应商基本信息：
    • 供应商名称（必填）
    • 联系人信息
    • 银行账户
    • 税务信息
    • 合作状态

6.2 供应商评估

评估指标：
  1. 交货准时率
  2. 产品质量合格率
  3. 价格竞争力
  4. 售后服务响应
  5. 合作稳定性

7. 采购报表查询

7.1 采购统计报表

报表路径：
  1. 点击菜单：采购管理 → 报表查询
  2. 选择统计期间和统计维度
  3. 系统生成采购分析报表：
    • 采购金额统计
    • 供应商采购排行
    • 物料采购分析
    • 采购趋势分析

7.2 供应商对账

对账操作：
  1. 选择供应商和对账期间
  2. 系统显示该期间的所有交易记录
  3. 核对采购订单、入库单、发票信息
  4. 生成对账单，双方确认

8. 常见问题解答

Q: 采购申请和采购订单有什么区别？

A: 采购申请是内部需求申请，需要审批；采购订单是向供应商发出的正式订单。

Q: 如何快速复制已有的采购订单？

A: 在采购订单查询界面，选择要复制的订单，点击"复制订单"按钮。

Q: 到货数量与订单数量不一致怎么办？

A: 在入库时填写实际数量，系统会记录差异，需要联系供应商处理。

Q: 采购订单作废后会影响什么？

A: 作废订单会取消后续的入库和付款操作，需要重新创建订单。

Q: 如何查看采购订单的处理进度？

A: 在采购订单查询界面，可以通过状态筛选查看订单的当前处理进度。

9. 操作注意事项

供应商选择：选择合格的供应商，确保物料质量和交货准时。
价格谈判：采购前进行充分的价格比较和谈判，控制采购成本。
合同管理：重要采购应签订正式合同，明确双方权利义务。
及时跟进：采购过程中要及时跟进订单状态，确保按时交货。
