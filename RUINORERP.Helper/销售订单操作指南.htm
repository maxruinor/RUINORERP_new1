销售订单操作指南

1. 销售订单创建

1.1 进入销售订单界面

操作路径：
  1. 登录系统后，点击顶部菜单：销售管理
  2. 在左侧功能树中选择：销售订单
  3. 点击按钮
界面说明：

销售订单编辑界面包含以下主要区域：

  • 订单头信息：客户信息、订单日期、交货日期等
  • 商品明细：商品列表、数量、单价、金额
  • 订单金额：小计、折扣、税额、总额
  • 操作按钮：保存、审核、打印等

1.2 填写订单基本信息

必填字段说明：
  1. 客户名称：点击搜索按钮选择客户（带*号必填）
  2. 订单日期：系统默认当前日期，可修改
  3. 交货日期：预计交货时间（重要提醒字段）
  4. 销售部门：自动带出当前用户所在部门
  5. 业务员：自动带出当前登录用户
提示：如果客户不存在，需要先在"客户管理"中创建客户资料。

1.3 添加商品明细

添加商品步骤：
  1. 点击商品明细区域的按钮
  2. 在弹出的商品选择窗口中搜索或选择商品
  3. 输入销售数量（必须大于0）
  4. 输入销售单价（系统会根据价格策略自动带出）
  5. 系统自动计算金额 = 数量 × 单价
  6. 点击确定添加到订单明细
重要：商品单价会根据客户等级和价格策略自动计算，如需修改需要相应权限。

1.4 订单金额计算

金额计算规则：
  1. 商品金额小计：所有商品金额总和
  2. 折扣金额：如有折扣，输入折扣率或折扣金额
  3. 税额：根据税率自动计算
  4. 订单总额：小计 - 折扣 + 税额

1.5 保存订单

保存操作：
  1. 确认所有信息填写正确后，点击按钮
  2. 系统会验证必填字段和数据有效性
  3. 保存成功后，订单状态变为"已保存"
  4. 系统自动生成订单编号（如：SO202501030001）

2. 订单审核流程

2.1 提交审核

审核提交：
  1. 订单保存后，点击按钮
  2. 系统根据订单金额判断审核流程：
    • 金额 ≤ 500元：自动审核通过
    • 金额 > 500元：需要指定人员审核
  3. 提交后订单状态变为"待审核"

2.2 审核人员操作

审核操作路径：
  1. 审核人员登录系统后，在工作台看到待审核订单提醒
  2. 点击提醒进入订单审核界面
  3. 查看订单详细信息
  4. 点击或
  5. 如不通过，需要填写不通过原因

2.3 审核结果处理

审核通过后：
  1. 订单状态变为"已审核"
  2. 系统自动通知仓库人员准备出库
  3. 销售人员在订单查询界面可以看到审核状态
审核不通过：
  1. 订单状态变为"审核不通过"
  2. 原销售人员收到通知，需要修改订单
  3. 修改后重新提交审核

3. 订单修改和作废

3.1 修改已保存订单

修改步骤：
  1. 在订单查询界面找到需要修改的订单
  2. 双击订单记录进入编辑界面
  3. 修改相关信息
  4. 点击按钮
注意：已提交审核或已审核的订单不能直接修改，需要先反审核。

3.2 反审核操作

反审核条件：
  1. 订单未生成出库单
  2. 具有反审核权限的用户
  3. 点击按钮
  4. 填写反审核原因
  5. 反审核后订单状态回到"已保存"

3.3 订单作废

作废条件：
  1. 订单未执行任何后续操作（出库、收款等）
  2. 具有订单作废权限
  3. 点击按钮
  4. 填写作废原因
  5. 作废后订单不能再进行任何操作

4. 订单查询和跟踪

4.1 订单查询界面

查询操作：
  1. 进入：销售管理 → 销售订单查询
  2. 设置查询条件：日期范围、客户、状态等
  3. 点击按钮
  4. 系统显示符合条件的订单列表

4.2 订单状态说明

状态说明可执行操作
已保存订单已创建但未提交审核修改、提交审核、删除
待审核已提交等待审核查看（只能审核人员操作）
已审核审核通过，等待出库查看、反审核、生成出库单
部分出库部分商品已出库查看、继续出库
已完成全部商品已出库查看、生成应收款
已作废订单已作废查看

5. 常见问题解答

Q: 如何快速复制已有的订单？

A: 在订单查询界面，选择要复制的订单，点击"复制订单"按钮，系统会创建相同客户和商品的新订单。

Q: 订单提交后发现有错误怎么办？

A: 如果订单还在"待审核"状态，可以联系审核人员退回修改。如果已审核，需要进行反审核操作。

Q: 如何查看订单的处理历史？

A: 在订单详情界面，点击"操作日志"标签可以查看订单的所有操作记录。

Q: 订单金额超过权限怎么办？

A: 系统会根据金额自动分配审核人员，如果超过个人权限，需要上级主管审核。

Q: 如何打印订单？

A: 在订单编辑或查询界面，点击"打印"按钮选择打印模板，可以打印订单确认书或送货单。

6. 操作注意事项

数据准确性：订单信息一旦提交将影响后续的出库和收款，请务必确保信息准确。
权限控制：不同用户角色有不同的操作权限，如遇权限问题请联系管理员。
时效性：订单处理要及时，避免影响客户交货和公司业务流程。
