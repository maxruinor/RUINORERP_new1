# 供应商（厂商）

## 说明
采购订单的供应商，即向谁采购货物或服务。

## 业务意义
供应商是采购业务的核心关联对象，选择正确的供应商关系到采购价格、交货期、产品质量等关键因素。系统通过供应商档案存储所有合作供应商的信息，包括联系方式、价格策略、付款条件等。

## 数据来源
供应商信息来源于基础资料中的供应商档案（tb_CustomerVendor），需要先在供应商档案中维护供应商信息，才能在采购订单中选择。

## 使用方法
1. 点击供应商下拉框
2. 在弹出的供应商列表中选择
3. 系统自动带出供应商的联系人、电话、默认付款条件等信息
4. 可以修改供应商（需在未审核状态下）

## 相关字段
- **联系人** (`txtContactPerson`): 供应商的联系人
- **联系电话** (`txtContactPhone`): 供应商的联系电话
- **结算账户** (`cmbPayeeInfoID`): 付款时的收款账户

## 业务规则
1. 供应商必须选择，不能为空
2. 供应商需要是审核状态才能选择
3. 选择供应商后，系统会自动应用该供应商的价格策略
4. 更换供应商需要重新计算价格

## 常见问题

### Q1: 供应商无法选择？

A: 检查供应商档案是否已审核，未审核的供应商不能使用。

### Q2: 如何查看供应商的价格？

A: 在供应商档案中可以查看该供应商的不同产品的价格策略。

### Q3: 供应商信息变更后，采购订单会更新吗？

A: 不会更新。采购订单中的供应商信息是下单时的快照，不会随供应商信息变更而更新。
