# 预交日期

## 说明
供应商承诺的交货日期，即预计货物到达的日期。

## 业务意义
预交日期是采购订单的重要时间节点，用于：
- 跟踪采购订单的执行进度
- 提前安排仓库收货准备
- 超期预警和催货提醒
- 影响销售订单的交期承诺

## 数据规则
- **默认值**: 根据采购数量和供应商交期自动计算
- **可修改**: 可以手动设置
- **格式**: 日期时间类型

## 使用方法
1. 在订单明细设置后，系统自动计算预交日期
2. 如需修改，点击日期选择器
3. 选择需要的交货日期
4. 可以为每批设置不同的预交日期（分批交货）

## 业务规则
1. 预交日期不能早于采购日期
2. 预交日期影响MRP运算的结果
3. 超期未交货系统会自动预警
4. 修改预交日期需要记录原因

## 常见问题

### Q1: 如何查看订单是否按时交货？

A: 在采购订单列表中可以查看预交日期和实际入库日期的对比，超期的会有颜色标识。

### Q2: 预交日期和实际到货日期不同以哪个为准？

A: 两个日期分别记录不同含义。预交日期是计划日期，实际到货日期是实绩日期。系统会根据实际到货日期更新订单状态。
