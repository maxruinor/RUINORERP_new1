# 采购日期

## 说明
采购订单的创建日期，即下单的日期。

## 业务意义
采购日期是采购业务的重要时间标识，用于：
- 确定采购订单的账期起算日期
- 统计特定时期的采购情况
- 与供应商对账的时间依据

## 数据规则
- **默认值**: 系统默认为当前日期
- **可修改**: 可以修改为其他日期
- **限制**: 不能晚于当前日期太多（可设置控制）

## 使用方法
1. 系统默认填充当前日期
2. 如需修改，点击日期选择器
3. 选择需要的日期

## 业务规则
1. 采购日期影响付款账期的计算
2. 采购日期影响成本核算的期间归属
3. 跨期间的采购订单需要特别处理

## 常见问题

### Q1: 采购日期可以修改吗？

A: 未审核的订单可以修改；已审核的订单不能直接修改，需要先取消审核。

### Q2: 采购日期与入库日期有什么区别？

A: 采购日期是下单日期，入库日期是实际收到货物的日期。两者可能不同，特别是分批交货的情况。
