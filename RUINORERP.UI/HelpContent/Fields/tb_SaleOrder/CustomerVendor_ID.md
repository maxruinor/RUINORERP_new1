# 客户

## 字段说明

客户字段用于指定销售订单的购买方，是订单的最基本信息之一。选择客户后，系统会自动应用该客户的价目表、信用额度、付款条件等信息。

## 业务意义

### 客户管理
- **客户档案**：系统维护完整的客户信息，包括基本信息、联系方式、财务信息等
- **客户分类**：客户按行业、地区、等级等进行分类，便于管理和统计
- **客户价目表**：不同客户可以有不同的价格体系

### 信用管理
- **信用额度**：每个客户都有设定的信用额度，订单金额不能超过信用额度
- **信用控制**：系统会自动检查客户信用状态，逾期未付款的客户会限制新增订单
- **信用评分**：系统根据客户的付款记录自动计算信用评分

### 关联信息
选择客户后，系统会自动带出以下信息：
- 客户的默认收款账户
- 客户的付款条件
- 客户的专属价格
- 客户的交货地址
- 客户的业务员

## 使用说明

### 选择客户
1. 点击"客户"下拉框（`cmbCustomerVendor_ID`）
2. 在弹出的客户选择窗口中搜索客户
3. 可以按客户名称、编码、联系人等信息搜索
4. 选中客户后点击"确定"

### 新建客户
1. 在客户选择窗口点击"新建"按钮
2. 填写客户基本信息
3. 保存客户档案
4. 返回订单界面，选择新建的客户

### 查看客户详情
1. 双击客户下拉框
2. 系统打开客户详情界面
3. 可以查看客户的完整信息

## 重要提示

### 信用控制
- **信用额度检查**：订单金额超过信用额度时，系统会阻止保存
- **逾期账款检查**：有逾期未付款的客户限制新增订单
- **信用审批**：超过信用额度但确实需要时，可以申请信用审批

### 价格体系
- **价目表优先级**：客户专属价格 > 客户分类价格 > 标准价格
- **价格审批**：修改价格超过允许范围时需要审批
- **历史价格**：系统记录该客户的历史成交价，方便参考

### 特殊客户
- **平台客户**：电商平台订单的客户，不计入信用额度
- **内部客户**：公司内部使用的客户，特殊处理
- **样品客户**：仅用于样品赠送的客户

## 常见问题

### Q1: 选择客户后价格不对？

A: 检查以下几点：
1. 客户是否有专属价目表
2. 价目表是否有效（在有效期内）
3. 价格是否被修改过

### Q2: 客户下拉框中找不到客户？

A: 可能原因：
1. 客户被禁用了，联系管理员启用
2. 搜索条件不正确，尝试其他搜索方式
3. 客户确实不存在，需要新建

### Q3: 如何查询客户的信用额度？

A: 打开客户详情界面，可以查看：
- 当前信用额度
- 已使用额度
- 剩余可用额度
- 逾期账款金额

### Q4: 客户信用额度超限怎么办？

A: 有以下解决方案：
1. 提醒客户付款，释放信用额度
2. 申请临时提高信用额度（需要审批）
3. 要求客户预付部分款项

## 相关字段

- **收款账户**：客户指定的收款银行账户
- **业务员**：负责该客户的业务员
- **付款状态**：该订单的付款状态
- **客户PO号**：客户方的采购订单号
