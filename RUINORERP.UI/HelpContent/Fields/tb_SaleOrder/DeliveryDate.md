# 交货日期

## 说明
产品计划交付给客户的日期。交货日期用于安排生产计划和物流发货。

## 使用方法
1. 点击交货日期输入框
2. 从日期选择器中选择日期
3. 或手动输入日期(格式: yyyy-MM-dd)

## 注意事项
1. **必填项**: 交货日期为必填字段
2. **日期验证**: 交货日期必须晚于订单日期
3. **生产周期**: 交货日期应考虑产品生产周期
4. **节假日**: 请避开节假日安排交货
5. **紧急订单**: 如果交货日期过近,系统会显示警告提示

## 快捷键
- `F4`: 快速设置交货日期(加3天、加7天等)
- `F1`: 查看帮助

## 常见问题

### Q: 交货日期可以设置为周末吗?
A: 可以,但建议与客户确认是否接受周末收货。

### Q: 如何批量修改交货日期?
A: 订单审核后,可以使用"批量修改交货日期"功能。
