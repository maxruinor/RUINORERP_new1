# 订单日期

## 说明
订单创建日期，默认为当前系统日期。订单日期会影响价格策略、交货计划等。

## 使用方法
1. 系统自动填充为当前日期
2. 可以手动修改为其他日期
3. 点击日期选择器选择日期

## 注意事项
1. **日期限制**: 订单日期不能早于业务开始日期
2. **价格策略**: 部分产品的价格会根据订单日期调整
3. **报表统计**: 订单日期是报表统计的重要依据
4. **历史订单**: 可以通过订单日期筛选历史订单

## 快捷键
- `F4`: 快速选择常用日期(今天、本周、本月等)
- `F1`: 查看帮助
