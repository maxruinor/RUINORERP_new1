# 总成本

## 字段说明

总成本字段显示销售订单中所有产品的成本总和，包括产品基础成本和定制成本。总成本是计算毛利的重要依据。

## 业务意义

### 成本构成
总成本由以下几部分组成：

```
总成本 = 产品基础成本 × 数量 + 定制成本
```

#### 产品基础成本
- **标准产品成本**：产品档案中维护的标准成本
- **实际成本**：采用先进的成本核算方法计算的实际成本
- **成本核算方法**：支持先进先出（FIFO）、加权平均、移动平均等

#### 定制成本
- **特殊材料成本**：定制产品所需的特殊材料费用
- **额外工时成本**：定制产品额外的人工费用
- **设计费用**：定制产品的设计开发费用
- **其他费用**：模具费、测试费等

### 毛利计算
```
毛利 = 销售金额 - 总成本
毛利率 = 毛利 / 销售金额 × 100%
```

### 经营决策
- **定价参考**：根据总成本确定合理的销售价格
- **利润分析**：评估订单的盈利能力
- **成本控制**：监控成本波动，及时发现问题

## 成本核算方法

### 先进先出（FIFO）
- **说明**：先采购/生产的产品先出库
- **适用场景**：产品价格波动较小
- **特点**：接近市场实际成本

### 加权平均
- **说明**：按库存数量和成本计算平均成本
- **适用场景**：产品价格波动较大
- **特点**：成本平稳，不受单次影响

### 移动平均
- **说明**：每次入库后重新计算平均成本
- **适用场景**：需要实时反映成本变化
- **特点**：最接近当前市场成本

## 使用说明

### 自动计算
- 总成本由系统自动计算
- 添加订单明细后自动更新
- 修改产品或数量后自动重新计算

### 手工调整
- 一般不建议手工调整
- 特殊情况下（如成本核算错误）可以调整
- 需要管理员权限
- 调整后需要审批

### 定制成本
- 定制订单需要在订单明细中填写定制成本
- 定制成本会自动计入总成本
- 定制成本不向客户展示

## 成本分析

### 成本差异分析
系统会自动对比：
- **计划成本**：预计的成本
- **实际成本**：实际发生的成本
- **差异金额**：计划与实际的差异
- **差异原因**：分析差异产生的原因

### 成本趋势分析
- **产品成本趋势**：监控单个产品成本的变化
- **总体成本趋势**：监控整体成本水平
- **成本构成分析**：分析材料、人工、制造费用的占比

## 重要提示

### 成本准确性
- **及时更新**：产品成本发生变化时及时更新档案
- **定期盘点**：定期进行库存盘点，调整账面成本
- **数据核对**：定期与财务成本数据核对

### 成本保密
- **查看权限**：成本信息只有授权人员才能查看
- **打印控制**：打印单据时可以隐藏成本信息
- **客户展示**：成本信息不向客户展示

### 成本控制
- **预算管理**：为产品设置标准成本预算
- **差异报警**：实际成本超过预算时自动报警
- **成本优化**：持续优化生产流程，降低成本

## 常见问题

### Q1: 总成本是如何计算的？

A: 总成本 = 产品成本 × 数量 + 定制成本。产品成本按成本核算方法（FIFO、加权平均等）自动计算。

### Q2: 为什么总成本不对？

A: 可能原因：
1. 产品成本未更新，请检查产品档案
2. 成本核算方法不正确，请检查系统配置
3. 定制成本未填写（定制订单）
4. 系统缓存问题，请刷新数据

### Q3: 可以手工修改总成本吗？

A: 一般不建议手工修改。如果确实需要：
1. 需要管理员权限
2. 需要填写修改原因
3. 需要经过审批
4. 系统会记录修改日志

### Q4: 如何降低订单成本？

A: 可以从以下方面优化：
1. **优化产品设计**：简化产品结构，减少材料
2. **优化生产流程**：提高效率，降低人工成本
3. **批量采购**：降低材料采购成本
4. **供应商管理**：选择性价比高的供应商

### Q5: 成本数据会影响什么？

A: 成本数据影响：
1. **毛利分析**：计算订单利润率
2. **财务核算**：生成销售成本凭证
3. **经营决策**：定价、产品组合、市场策略
4. **绩效评估**：评估业务员和工厂的业绩
