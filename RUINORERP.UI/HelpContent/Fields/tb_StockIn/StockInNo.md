# 入库单号

## 说明
采购入库单的唯一标识编号，由系统自动生成，不可修改。

## 编号规则
入库单号采用编码规则自动生成，格式: `PI + 年月 + 4位流水号`

例如: `PI2024010001`

### 示例
| 编号 | 说明 |
|------|------|
| PI2024010001 | 2024年1月的第1个入库单 |
| PI2024010256 | 2024年1月的第256个入库单 |
| PI2024020001 | 2024年2月的第1个入库单 |

## 使用方法
1. **自动生成**: 审核入库单时自动生成
2. **手动修改**: 不支持手动修改
3. **查询引用**: 在其他单据中通过入库单号引用

## 业务用途
- 库存台账查询
- 应付账款关联
- 成本核算依据
- 采购跟踪

## 注意事项
1. **唯一性**: 入库单号在系统中唯一
2. **不可修改**: 生成后不允许修改
3. **编码规则**: 可在系统配置中修改编码规则
