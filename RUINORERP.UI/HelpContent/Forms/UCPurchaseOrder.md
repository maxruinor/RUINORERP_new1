# 采购订单窗体

## 窗体概述

采购订单窗体用于创建和管理采购订单，是企业与供应商之间的正式采购合同凭证。该窗体支持标准采购、外协加工、分批交货等多种业务场景，是采购业务的核心操作界面。

采购订单与采购需求、销售订单和生产计划紧密关联。系统可以根据MRP运算结果或安全库存自动生成采购建议，用户可以快速将采购建议转化为采购订单。同时，采购订单可以关联销售订单，为销售订单中涉及的外购物料进行采购。

## 窗体布局

### 主信息区
- **供应商信息**：供应商、采购联系人、结算账户
- **订单信息**：采购单号、订单日期、预计到货日期
- **付款信息**：付款方式、付款期限、币别
- **运输信息**：运输方式、到货仓库、运费承担
- **审批信息**：审批结果、审批意见

### 明细区（表格）
- 产品明细列表
- 采购数量、单价、金额
- 税率、税额
- 交货日期、分批数量

### 汇总区
- 总数量、总金额
- 税额、运费
- 优惠金额、应付金额

## 基本操作流程

### 新建采购订单

1. 点击工具栏"新建"按钮或按 `Ctrl+N`
2. 选择供应商（`cmbSupplier_ID`）
3. 选择到货仓库（`cmbStore_ID`）
4. 添加订单明细（产品、数量、价格）
5. 设置交货日期
6. 保存订单（`Ctrl+S`）

### 从采购建议生成订单

1. 在采购建议列表中选择要执行的建议
2. 点击"生成订单"按钮
3. 系统自动创建采购订单并带出建议内容
4. 确认供应商、价格等信息
5. 保存订单

### 审核订单

1. 打开采购订单
2. 检查订单信息是否正确
3. 点击"审核"按钮
4. 审核后订单进入执行阶段

### 取消审核

1. 打开已审核的订单
2. 点击"取消审核"按钮
3. 订单变为编辑状态，可以修改

### 删除订单

1. 查询要删除的订单（未审核状态）
2. 点击"删除"按钮
3. 确认删除操作

## 重要业务场景

### 场景1：标准采购

**业务场景**：向供应商采购原材料或成品

**操作步骤**：
1. 选择供应商
2. 选择到货仓库
3. 添加采购产品明细
4. 填写采购数量和单价
5. 设置预计到货日期
6. 保存并审核订单

**系统处理**：
- 订单进入执行阶段
- 可以在到货后生成入库单
- 入库后生成应付账款

### 场景2：分批交货

**业务场景**：订单数量较大，供应商分多次交货

**操作步骤**：
1. 创建采购订单，录入总数量
2. 在明细中设置每批交货数量和日期
3. 审核订单
4. 第一批到货：创建入库单，录入到货数量
5. 系统更新订单的未交货数量
6. 后续批次重复入库操作

**分批交货说明**：
- 明细行可以设置多个计划交货日期
- 每批到货可以创建独立的入库单
- 系统跟踪每批的交货状态
- 全部到货后订单自动关闭

### 场景3：外协加工

**业务场景**：委托供应商进行加工处理

**操作步骤**：
1. 勾选"外协加工"复选框（`chkIsOuterProcessing`）
2. 选择加工供应商
3. 添加需要加工的产品和数量
4. 设置加工要求和交货日期
5. 保存订单

**外协加工说明**：
- 外协加工需要先发料（创建发料单）
- 加工完成后需要收货入库
- 加工费用需要单独结算

### 场景4：关联销售订单

**业务场景**：为销售订单中的外购物料进行采购

**操作步骤**：
1. 在采购订单中选择关联的销售订单
2. 系统自动带出需要采购的产品明细
3. 确认采购数量和价格
4. 保存订单

**关联说明**：
- 采购入库后可以自动关联到销售订单
- 方便跟踪销售订单的采购进度
- 可以查看采购成本对销售毛利的影响

### 场景5：采购变更

**业务场景**：采购订单签订后需要修改

**操作步骤**：
1. 查询要变更的订单（已审核状态）
2. 点击"变更"按钮
3. 修改明细内容（数量、价格、交期等）
4. 填写变更原因
5. 提交审批（如果需要）

**变更说明**：
- 已审核的订单需要通过变更来修改
- 变更会生成变更记录
- 变更后需要重新审批

## 关键字段说明

### 供应商信息字段

| 字段 | 控件 | 业务意义 | 使用场景 |
|-----|------|---------|---------|
| 供应商 | `cmbSupplier_ID` | 采购供应商 | 所有采购必填 |
| 联系人 | `txtContactPerson` | 供应商联系人 | 联系送货事宜 |
| 联系电话 | `txtContactPhone` | 供应商电话 | 紧急联系 |
| 结算账户 | `cmbAccount_ID` | 付款收款账户 | 付款时使用 |

### 订单信息字段

| 字段 | 控件 | 业务意义 | 使用场景 |
|-----|------|---------|---------|
| 采购单号 | `txtPurOrderNo` | 订单唯一标识 | 系统自动生成 |
| 订单日期 | `dtpOrderDate` | 订单创建日期 | 默认当天 |
| 预计到货日期 | `dtpExpectedDate` | 计划交货日期 | 供应商承诺的交货时间 |
| 有效截止日期 | `dtpValidDate` | 订单有效期 | 超过后订单自动失效 |

### 付款信息字段

| 字段 | 控件 | 业务意义 | 使用场景 |
|-----|------|---------|---------|
| 付款方式 | `cmbPayType_ID` | 付款方式 | 现金、转账、承兑等 |
| 付款期限 | `cmbPayPeriod` | 付款账期 | 月结30天、60天等 |
| 币别 | `cmbCurrency_ID` | 结算货币 | 人民币、美元等 |
| 汇率 | `txtExchangeRate` | 外币汇率 | 使用外币时 |

### 运输信息字段

| 字段 | 控件 | 业务意义 | 使用场景 |
|-----|------|---------|---------|
| 运输方式 | `cmbShippingWay` | 运输方式 | 快递、物流、自提等 |
| 到货仓库 | `cmbStore_ID` | 收货仓库 | 必须指定 |
| 运费 | `txtFreight` | 运费金额 | 供应商承担或买方承担 |
| 运费方 | `cmbFreightPayer` | 运费付款方 | 采购方或供应商 |

### 特殊选项字段

| 字段 | 控件 | 业务意义 | 使用场景 |
|-----|------|---------|---------|
| 外协加工 | `chkIsOuterProcessing` | 是否为外协加工 | 外协加工勾选 |
| 关联销售 | `chkIsRelatedSO` | 是否关联销售订单 | 为销售订单采购时 |
| 已定价 | `chkIsPriced` | 是否已确认价格 | 价格已确认时 |
| 含税 | `chkIsTaxIncluded` | 价格是否含税 | 默认含税 |

## 订单明细表格

### 添加产品明细

1. 点击明细表格中的"新增"按钮
2. 选择产品（从产品档案中选择）
3. 输入采购数量
4. 输入或选择单价（系统会应用供应商价格）
5. 系统自动计算金额和税额

### 修改产品明细

1. 在表格中双击要修改的行
2. 修改数量、价格等信息
3. 点击"确定"保存修改

### 删除产品明细

1. 选中要删除的行
2. 点击"删除"按钮
3. 确认删除操作

### 分批交货设置

在明细行的"交货日期"列可以设置多个交货计划：
1. 双击明细行打开编辑对话框
2. 在"交货计划"中添加多行
3. 每行设置交货日期和数量
4. 保存后系统按计划跟踪交货

## 状态流转

### 订单状态

| 状态 | 说明 | 可执行操作 |
|-----|------|-----------|
| 新建 | 订单刚创建，未审核 | 编辑、审核、删除 |
| 已审核 | 订单已审核，等待执行 | 变更、取消审核、生成入库 |
| 部分到货 | 已部分到货入库 | 继续入库、变更 |
| 已完成 | 全部到货入库 | 查询、打印 |
| 已取消 | 订单已取消 | 查询（不可编辑） |
| 已关闭 | 订单已手动关闭 | 查询（不可编辑） |

### 订单类型

| 类型 | 说明 |
|-----|------|
| 标准采购 | 常规采购业务 |
| 外协加工 | 委托加工 |
| 受托代销 | 代销商品 |
| 采购退货 | 退货补货 |

## 常见业务问题

### Q1：采购订单可以修改吗？

A：已审核的订单不能直接修改，需要先取消审核。建议使用"订单变更"功能，这样可以保留变更历史记录。

### Q2：采购数量与入库数量不一致？

A：系统支持部分入库。如果实际到货数量少于订单数量，可以先入库实际数量，剩余数量可以后续到货后继续入库，或与供应商协商处理。

### Q3：如何查看采购订单的入库进度？

A：在采购订单详情界面，可以查看该订单的入库明细，包括已入库数量、未入库数量、入库单号等信息。

### Q4：供应商价格如何设置？

A：在供应商档案中可以为不同产品设置不同的采购价格。创建采购订单时，系统会自动应用对应的价格。

### Q5：采购订单需要审批吗？

A：可以在系统参数中设置采购订单的审批流程。大额采购或特殊类型的采购可能需要多级审批。

## 打印功能

### 打印采购订单

1. 打开采购订单
2. 点击工具栏"打印"按钮
3. 选择打印模板（采购订单、采购合同）
4. 预览打印效果
5. 确认打印

### 导出采购订单

1. 打开采购订单查询界面
2. 选择要导出的订单
3. 点击"导出"按钮
4. 选择导出格式（Excel、PDF）
5. 保存文件

## 注意事项

1. **供应商价格**：创建订单时系统会自动应用供应商的价格策略
2. **库存控制**：到货数量不能超过订单数量太多（可设置容差）
3. **付款控制**：注意供应商的付款账期，及时安排付款
4. **成本核算**：入库成本按采购价格或成本方法计算
5. **数据保存**：离开窗体前请确保已保存数据，否则修改会丢失

## 错误处理

### 常见错误及解决方案

| 错误提示 | 原因 | 解决方案 |
|---------|------|---------|
| 供应商未设置价格 | 产品未设置供应商价格 | 在供应商价格中维护价格 |
| 超过有效期 | 订单超过有效截止日期 | 修改有效期或重新创建 |
| 产品已停产 | 产品标记为停产 | 选择其他产品 |
| 仓库不存在 | 仓库未在系统中维护 | 在基础资料中维护仓库 |
