# 库存盘点单窗体

## 窗体概述

库存盘点单窗体用于进行库存盘点操作，确保库存账实相符。盘点是库存管理的重要环节，通过定期或不定期的盘点，可以发现库存差异、分析差异原因并进行相应的账务调整。

系统支持多种盘点方式，包括定期盘点、动态盘点和盲盘等。盘点单创建后，可以生成分区域的盘点任务，盘点人员实地清点后录入实际数量，系统自动计算差异并生成调整建议。

## 窗体布局

### 主信息区
- **盘点信息**：盘点单号、盘点日期、盘点类型
- **仓库信息**：盘点仓库、盘点范围
- **计划信息**：计划开始日期、计划结束日期
- **状态信息**：盘点状态、差异状态

### 明细区（表格）
- 产品信息
- 账面数量
- 盘点数量
- 差异数量
- 差异金额

### 汇总信息区
- 盘点品种数
- 差异品种数
- 盘盈数量、金额
- 盘亏数量、金额

## 基本操作流程

### 创建盘点单

1. 点击"新建"按钮
2. 选择盘点仓库
3. 设置盘点范围（全库、分类、库位）
4. 设置盘点类型（定期、动态、盲盘）
5. 生成盘点明细（系统带出账面库存）
6. 保存盘点单

### 实地盘点

1. 打印盘点表或使用PDA盘点
2. 盘点人员到仓库实地清点
3. 录入实际盘点数量
4. 系统自动计算差异

### 差异审核

1. 审核人员审核差异
2. 确认差异原因
3. 确认是否调整
4. 审核后系统自动调整库存

## 重要业务场景

### 场景1：定期全面盘点

**业务场景**：定期对所有仓库进行全面盘点

**操作步骤**：
1. 创建盘点单，选择全部仓库
2. 设置盘点类型为"定期盘点"
3. 系统生成所有产品的盘点明细
4. 实地盘点并录入数量
5. 审核差异并调整库存

### 场景2：动态盘点

**业务场景**：日常工作中随时进行盘点

**操作步骤**：
1. 设置盘点类型为"动态盘点"
2. 可以随时开始盘点
3. 不影响正常出入库业务
4. 盘点结果随时可以审核

### 场景3：盲盘

**业务场景**：不显示账面数量，避免盘点人员作弊

**操作步骤**：
1. 设置盘点类型为"盲盘"
2. 盘点明细不显示账面数量
3. 盘点人员只录入实际数量
4. 盘点完成后系统计算差异

## 关键字段说明

### 盘点信息字段

| 字段 | 控件 | 业务意义 | 使用场景 |
|-----|------|---------|---------|
| 盘点单号 | `txtCheckNo` | 盘点单唯一标识 | 系统自动生成 |
| 盘点日期 | `dtpCheckDate` | 盘点日期 | 默认当天 |
| 盘点类型 | `cmbCheckType` | 盘点方式 | 定期、动态、盲盘 |

### 盘点范围字段

| 字段 | 控件 | 业务意义 | 使用场景 |
|-----|------|---------|---------|
| 盘点仓库 | `cmbStore_ID` | 要盘点的仓库 | 必选 |
| 盘点范围 | `cmbCheckRange` | 全库/分类/库位 | 可选 |
| 产品分类 | `cmbProdCategory` | 特定分类 | 按分类盘点时 |

### 状态字段

| 字段 | 状态 |
|-----|------|
| 盘点状态 | 待盘点、盘点中、已完成 |
| 差异状态 | 无差异、待审核、已审核 |

## 盘点明细

### 账面数量

- **来源**：系统根据库存台账计算
- **时点**：盘点创建时的库存数量
- **包括**：可用库存、占用库存、冻结库存

### 盘点数量

- **输入**：盘点人员录入
- **方式**：手工输入或扫描录入
- **验证**：可以设置允许负数等

### 差异计算

- **盘盈**：盘点数量 > 账面数量
- **盘亏**：盘点数量 < 账面数量
- **金额**：差异数量 × 单位成本

## 状态流转

### 盘点单状态

| 状态 | 说明 | 可执行操作 |
|-----|------|-----------|
| 新建 | 刚创建，待盘点 | 编辑、开始盘点 |
| 盘点中 | 正在实地盘点 | 录入盘点数量 |
| 已完成 | 盘点完成 | 审核差异 |
| 已审核 | 差异已审核，调整完成 | 查询、打印 |
| 已取消 | 盘点已取消 | 查询（不可编辑） |

## 差异处理

### 盘盈处理

1. 审核差异确认盘盈
2. 系统自动生成入库单
3. 增加库存数量
4. 计入营业外收入

### 盘亏处理

1. 审核差异确认盘亏
2. 系统自动生成出库单
3. 扣减库存数量
4. 计入管理费用

## 常见业务问题

### Q1：盘点时可以正常出入库吗？

A：动态盘点时可以进行出入库；定期盘点时通常会冻结出入库。

### Q2：差异必须调整吗？

A：是的，盘点差异需要经过审核并调整，以保持账实相符。

### Q3：可以只盘点部分产品吗？

A：可以，按产品分类或特定库位进行局部盘点。

### Q4：盲盘有什么好处？

A：盘点人员不知道账面数量，可以避免人情盘点，提高盘点准确性。

## 注意事项

1. **盘点准备**：盘点前尽量减少出入库
2. **盘点准确**：确保盘点数量准确
3. **及时调整**：差异要及时审核调整
