# 采购入库单窗体

## 窗体概述

采购入库单窗体用于处理采购订单到货后的入库操作，是采购业务流程的重要环节。该窗体支持从采购订单生成入库单、直接创建入库单、部分入库等多种业务场景，同时提供质量检验功能，确保入库商品的质量符合要求。

采购入库单与库存管理和应付账款管理紧密关联。入库操作会更新库存数量和成本，同时生成对应的应付账款。系统支持批次管理和序列号管理，能够精确跟踪每一批入库商品的信息。

## 窗体布局

### 主信息区
- **来源信息**：采购订单、入库单号、入库日期
- **供应商信息**：供应商、联系人、电话
- **仓库信息**：到货仓库、库位
- **财务信息**：币别、汇率、付款状态
- **检验信息**：检验状态、检验结论
- **审批信息**：审批结果、审批意见

### 明细区（表格）
- 产品明细列表
- 采购数量、入库数量、检验数量
- 单价、金额
- 批次号、序列号
- 检验结果

### 汇总区
- 总数量、总金额
- 合格率、税额

## 基本操作流程

### 从采购订单入库

1. 点击工具栏"新建"按钮
2. 选择要入库的采购订单
3. 系统自动带出订单明细
4. 确认入库数量（默认为订单剩余数量）
5. 进入质量检验环节（如需要）
6. 确认入库，审核单据
7. 系统自动更新库存和生成应付账款

### 直接创建入库单

1. 点击工具栏"新建"按钮
2. 不选择采购订单，直接创建
3. 选择供应商
4. 添加产品明细
5. 填写入库数量
6. 审核单据

### 部分入库

1. 选择采购订单
2. 录入本次实际入库数量（小于订单数量）
3. 系统记录已入库和未入库数量
4. 后续可以继续对同一订单进行入库

### 质量检验入库

1. 创建入库单后进入检验流程
2. 检验人员录入检验结果
3. 合格品确认入库
4. 不合格品进入退货或降级处理流程

## 重要业务场景

### 场景1：标准采购入库

**业务场景**：采购的货物到达，完成入库操作

**操作步骤**：
1. 选择采购订单
2. 系统带出订单明细
3. 确认入库数量
4. 审核入库单
5. 系统自动更新库存
6. 系统生成应付账款

**系统处理**：
- 库存数量增加
- 库存成本更新（按采购价格或成本方法）
- 应付账款生成
- 采购订单状态更新

### 场景2：部分入库

**业务场景**：订单数量较大，分多次到货

**操作步骤**：
1. 选择采购订单
2. 录入本次实际到货数量
3. 小于订单总数量
4. 审核入库单
5. 订单显示"部分入库"状态
6. 后续继续入库剩余数量

**系统处理**：
- 累计入库数量记录
- 采购订单自动跟踪未入库数量
- 可以继续从同一订单入库

### 场景3：来料检验

**业务场景**：到货商品需要经过质量检验

**操作步骤**：
1. 创建入库单
2. 提交质量检验
3. 检验员录入检验结果
4. 根据检验结果处理：
   - 合格：确认入库
   - 不合格：退货或降级接收
5. 完成入库操作

**检验结果说明**：
- **合格**：可以正常入库
- **不合格**：需要退货或索赔
- **让步接收**：可以入库但需要折价

### 场景4：外协加工入库

**业务场景**：外协加工完成后收回成品

**操作步骤**：
1. 关联外协加工采购订单
2. 系统带出加工产品明细
3. 确认入库数量
4. 审核入库单
5. 成品入库，增加库存
6. 加工费生成应付账款

### 场景5：采购退货

**业务场景**：入库后发现质量问题需要退货

**操作步骤**：
1. 查询要退货的入库单
2. 创建退货单或红字入库单
3. 选择退货产品和数量
4. 审核退货单
5. 系统扣减库存，冲减应付账款

## 入库明细表格

### 入库数量

- **默认值**：采购订单剩余数量
- **可修改**：可以小于订单数量（部分入库）
- **限制**：不能超过订单剩余数量
- **检验数量**：实际检验合格的数量

### 批次管理

- **批次号**：可以自动生成或手工指定
- **生产日期**：记录生产批次信息
- **有效期**：记录保质期信息
- **先进先出**：出库时按批次优先

### 序列号管理

- **序列号**：唯一标识每个产品
- **入库登记**：入库时录入或扫描
- **出库消耗**：出库时选择序列号
- **全程跟踪**：可以查询序列号的所有记录

## 状态流转

### 入库单状态

| 状态 | 说明 | 可执行操作 |
|-----|------|-----------|
| 新建 | 入库单刚创建，未审核 | 编辑、审核、删除 |
| 检验中 | 等待质量检验 | 查看、检验操作 |
| 已检验 | 检验完成，等待入库 | 确认入库、重新检验 |
| 已审核 | 已入库，库存已更新 | 取消审核、打印 |
| 已完成 | 已完成全部流程 | 查询、打印 |
| 已取消 | 入库单已取消 | 查询（不可编辑） |

### 检验状态

| 状态 | 说明 |
|-----|------|
| 无需检验 | 不需要质量检验 |
| 待检验 | 等待检验 |
| 检验中 | 正在进行检验 |
| 合格 | 检验合格，可以入库 |
| 不合格 | 检验不合格，需要处理 |
| 让步接收 | 折价接收不合格品 |

## 常见业务问题

### Q1：入库数量与订单数量不一致？

A：系统支持部分入库。如果实际到货数量少于订单数量，可以先入库实际数量，剩余数量等待后续到货或与供应商协商处理。

### Q2：入库后发现质量问题如何处理？

A：需要先进行退货处理，创建红字入库单冲减库存和应付账款，然后再与供应商协商赔偿事宜。

### Q3：如何查看入库单关联的应付账款？

A：在入库单详情界面可以查看该入库单生成的应付账款记录，包括金额、核销状态等信息。

### Q4：入库成本如何计算？

A：入库成本按采购价格计算，具体成本方法（先进先出、加权平均等）可以在系统参数中设置。

### Q5：如何处理赠品入库？

A：可以在入库明细中添加赠品，将数量和单价都设为0，赠品入库不生成应付账款。

## 打印功能

### 打印入库单

1. 打开入库单
2. 点击工具栏"打印"按钮
3. 选择打印模板（入库单、收货单）
4. 预览打印效果
5. 确认打印

### 打印检验报告

1. 打开入库单
2. 点击"打印检验报告"
3. 预览并打印

## 注意事项

1. **库存控制**：入库数量需要经过库存检查
2. **质量检验**：需要检验的商品必须检验合格后才能入库
3. **成本核算**：入库成本按采购价格或成本方法计算
4. **应付账款**：审核后自动生成应付账款
5. **数据保存**：离开窗体前请确保已保存数据

## 错误处理

### 常见错误及解决方案

| 错误提示 | 原因 | 解决方案 |
|---------|------|---------|
| 超过订单数量 | 入库数量超过订单剩余数量 | 减少入库数量或分批入库 |
| 仓库不存在 | 仓库未维护 | 在基础资料中维护仓库 |
| 检验未通过 | 检验不合格 | 处理不合格品或退货 |
| 批次已使用 | 批次号重复 | 使用新批次号 |
