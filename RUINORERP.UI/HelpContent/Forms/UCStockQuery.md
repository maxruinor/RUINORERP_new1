# 库存查询窗体

## 窗体概述

库存查询窗体用于查看当前库存情况和历史出入库记录，是库存管理的基础功能。该窗体提供多种查询维度，支持按仓库、产品、库位、批次等多种条件查询，能够帮助用户快速了解库存状况和进行库存分析。

库存查询不仅显示当前的库存数量，还提供可用库存、占用库存、冻结库存等细分信息，帮助用户准确判断是否可以进行销售或生产。同时，查询界面支持库存台账功能，可以查看每种产品的详细出入库记录。

## 窗体布局

### 查询条件区
- **仓库选择**：可以单选或多选仓库
- **产品选择**：可以按产品分类或具体产品查询
- **库位选择**：可以查询特定库位的库存
- **批次号**：可以查询特定批次的库存
- **库存状态**：可选正常、冻结、占用等

### 查询结果区（表格）
- 仓库、库位
- 产品编码、名称、规格
- 库存数量、可用数量、占用数量、冻结数量
- 单位、成本、金额
- 批次号、序列号

### 汇总信息区
- 总库存数量
- 总库存金额
- 库存品种数

## 基本操作流程

### 查询当前库存

1. 进入库存查询界面
2. 选择查询条件（仓库、产品等）
3. 点击"查询"按钮
4. 查看查询结果

### 查看库存明细

1. 在查询结果中双击某行
2. 弹出库存明细窗口
3. 查看该产品的详细出入库记录

### 导出库存数据

1. 查询出需要的库存数据
2. 点击"导出"按钮
3. 选择导出格式（Excel）
4. 保存文件

## 查询功能说明

### 按仓库查询

可以查看特定仓库的库存情况：
- 选择一个或多个仓库
- 系统显示所选仓库的库存汇总
- 可以对比不同仓库的库存差异

### 按产品查询

可以查看特定产品的库存分布：
- 输入产品编码或名称
- 系统显示该产品在不同仓库的库存
- 可以查看产品的库位分布

### 按库位查询

可以查看特定库位的库存：
- 选择仓库和库位
- 系统显示该库位的所有库存
- 适合仓库管理人员使用

### 按批次查询

可以查看特定批次的库存：
- 输入批次号
- 系统显示该批次的所有库存
- 适合需要跟踪批次的产品

### 库存状态筛选

可以按库存状态进行筛选：
- **正常库存**：可供使用的库存
- **占用库存**：已被订单占用的库存
- **冻结库存**：被冻结不能使用的库存
- **在途库存**：已发货但未收货的库存

## 关键字段说明

### 库存数量字段

| 字段 | 说明 | 业务意义 |
|-----|------|---------|
| 库存数量 | 当前总库存 | 仓库中的实际存货 |
| 可用库存 | 可以使用的库存 | 库存数量 - 占用数量 - 冻结数量 |
| 占用库存 | 已分配未出库 | 销售订单等已占用等待出库 |
| 冻结库存 | 被冻结的库存 | 冻结后不能使用 |

### 库存金额字段

| 字段 | 说明 | 计算方式 |
|-----|------|---------|
| 单位成本 | 平均成本 | 总金额 / 库存数量 |
| 库存金额 | 库存总价值 | 库存数量 × 单位成本 |

### 批次信息字段

| 字段 | 说明 | 用途 |
|-----|------|------|
| 批次号 | 批次唯一标识 | 跟踪特定批次 |
| 生产日期 | 批次生产日期 | 计算有效期 |
| 有效期至 | 批次到期日期 | 预警提醒 |

## 库存分析功能

### 库龄分析

系统支持按库龄分析库存结构：
- 按入库时间分组统计
- 分析呆滞库存
- 提醒及时处理

### ABC分析

根据库存价值和数量进行ABC分类：
- A类：高价值少数量
- B类：中等价值中等数量
- C类：低价值多数量

### 安全库存分析

对比当前库存与安全库存：
- 低于安全库存的产品提醒
- 高于最高库存的产品预警

## 常见业务问题

### Q1：库存数量与账面上不一致？

A：需要进行库存盘点，找出差异原因。检查是否有未记账的出入库单据。

### Q2：可用库存为负数是什么原因？

A：可能是因为先出库后入库导致，需要检查出入库顺序或进行调账处理。

### Q3：如何查询某产品的历史出入库记录？

A：双击查询结果中的产品行，在弹出的明细窗口中可以查看完整的出入库记录。

### Q4：库存金额是如何计算的？

A：库存金额按成本方法计算，可以在系统参数中查看和修改成本方法设置。

### Q5：如何查看在途库存？

A：在库存查询条件中选择"在途库存"状态，可以查看已发货未收货的库存。

## 快捷键

| 快捷键 | 功能 |
|-------|------|
| F1 | 显示帮助 |
| F5 | 刷新数据 |
| Ctrl+F | 打开查询条件 |
| Ctrl+E | 导出数据 |
| Enter | 执行查询 |

## 注意事项

1. **查询范围**：根据权限设置，可能只能看到部分仓库的库存
2. **数据实时性**：查询显示的是上一次统计后的数据，可能有轻微延迟
3. **成本显示**：成本金额需要相应权限才能查看
