# 财务管理模块

## 模块概述

财务管理模块是企业ERP系统的核心模块，负责记录和反映企业的经济业务活动，为企业的经营管理提供决策依据。该模块与销售、采购、库存、生产等业务模块紧密集成，实现业务与财务的一体化管理。

财务管理模块主要包括应收管理、应付管理、收付款管理、费用管理、凭证管理、成本核算和财务报表等功能。通过与业务模块的集成，系统能够自动生成应收应付账目、自动进行收款付款核销、自动生成财务凭证，大大提高了财务工作效率和数据的准确性。

## 核心业务流程

### 1. 应收账款流程

```
销售出库 → 生成应收 → 客户付款 → 收款核销 → 应收账款结清
```

**流程说明**：

1. 销售出库：销售出库后自动生成应收账款
2. 生成应收：系统生成应收记录，关联销售出库单
3. 客户付款：客户支付货款
4. 收款核销：收款单与应收记录自动核销
5. 应收账款结清：核销后应收账款余额为零

### 2. 应付账款流程

```
采购入库 → 生成应付 → 向供应商付款 → 付款核销 → 应付账款结清
```

**流程说明**：

1. 采购入库：采购入库后自动生成应付账款
2. 生成应付：系统生成应付记录，关联采购入库单
3. 向供应商付款：向供应商支付货款
4. 付款核销：付款单与应付记录自动核销
5. 应付账款结清：核销后应付账款余额为零

### 3. 费用报销流程

```
费用发生 → 费用报销 → 审批流程 → 付款处理 → 凭证生成
```

**流程说明**：

1. 费用发生：员工因公产生费用
2. 费用报销：员工提交报销申请
3. 审批流程：按设置的审批流程进行审批
4. 付款处理：审批通过后进行付款
5. 凭证生成：自动生成记账凭证

## 关键业务规则

### 应收账款规则

1. **应收生成**：销售出库时自动生成应收账款
2. **信用控制**：超过信用额度时需要特殊审批
3. **账期计算**：根据客户设定的账期计算到期日
4. **催收管理**：逾期自动提醒，跟踪催收过程
5. **坏账处理**：确认无法收回的应收账款需要计提坏账准备

### 应付账款规则

1. **应付生成**：采购入库时自动生成应付账款
2. **账期管理**：根据供应商设定的账期管理付款
3. **付款优先级**：按到期顺序安排付款计划
4. **折扣管理**：支持现金折扣的获取和计算
5. **对账管理**：定期与供应商进行应付对账

### 成本核算规则

1. **成本方法**：支持移动加权平均、先进先出等成本方法
2. **成本更新**：出库时实时计算成本
3. **成本分析**：提供成本差异分析和成本趋势分析
4. **期末处理**：期末进行成本结转和差异调整

### 凭证管理规则

1. **凭证生成**：业务单据自动生成记账凭证
2. **凭证审核**：凭证需要经过审核才能记账
3. **凭证修改**：审核前可以修改，审核后需要红冲
4. **凭证查询**：可以查询任意期间的记账凭证

## 常见业务场景

### 场景1：收款处理

**适用情况**：收到客户货款

**操作步骤**：
1. 进入收款管理界面
2. 点击"新建"创建收款单
3. 选择收款客户和收款账户
4. 输入收款金额和收款日期
5. 选择要核销的应收单据
6. 保存并审核收款单
7. 系统自动进行核销处理

### 场景2：付款处理

**适用情况**：向供应商支付货款

**操作步骤**：
1. 进入付款管理界面
2. 点击"新建"创建付款单
3. 选择付款供应商和付款账户
4. 输入付款金额和付款日期
5. 选择要核销的应付单据
6. 保存并审核付款单
7. 系统自动进行核销处理

### 场景3：费用报销

**适用情况**：员工报销差旅费等费用

**操作步骤**：
1. 员工提交费用报销申请
2. 填写费用类型、金额和说明
3. 上传相关发票和凭证
4. 按审批流程进行审批
5. 审批通过后进行付款
6. 自动生成记账凭证

### 场景4：客户对账

**适用情况**：与客户核对应收账款

**操作步骤**：
1. 进入应收账款对账界面
2. 选择要对账的客户
3. 系统显示该客户的应收明细
4. 与客户确认往来数据
5. 确认后形成对账记录
6. 处理差异事项

### 场景5：供应商对账

**适用情况**：与供应商核对应付账款

**操作步骤**：
1. 进入应付账款对账界面
2. 选择要对账的供应商
3. 系统显示该供应商的应付明细
4. 与供应商确认往来数据
5. 确认后形成对账记录
6. 处理差异事项

## 数据统计与报表

### 关键指标

- **应收账款周转率**：赊销收入/平均应收账款，反映收账速度
- **应收账款周转天数**：365/周转率，反映平均收账周期
- **应付账款周转天数**：365/(采购成本/平均应付账款)，反映付款速度
- **毛利率**：毛利/销售收入，反映盈利水平

### 常用报表

1. **应收账款账龄分析**：按账龄区间分析应收账款结构
2. **应付账款账龄分析**：按账龄区间分析应付账款结构
3. **资金预测表**：预测未来资金收支情况
4. **利润表**：反映企业一定期间的经营成果
5. **资产负债表**：反映企业一定时点的财务状况
6. **现金流量表**：反映企业现金收支情况

## 常见问题

### Q1：应收账款与实际不符怎么办？

A：首先需要检查销售出库单和收款单是否完整，然后进行核销处理。如果确实存在差异，需要调整对应的业务单据或手工调整应收账款。

### Q2：如何查看某客户的欠款情况？

A：在应收账款查询界面，选择客户后可以查看该客户所有的应收明细、已收金额、应收余额，以及账龄分析信息。

### Q3：付款时如何选择要核销的应付单据？

A：在付款单中可以选择自动核销或手工核销。自动核销按到期日期先后顺序核销，手工核销可以指定要核销的应付单据。

### Q4：成本核算方法可以修改吗？

A：成本核算方法可以在系统参数中设置，但修改后只对后续业务生效，已发生的成本不会自动调整。建议在系统上线前确定成本核算方法。

### Q5：如何生成财务报表？

A：在财务报表模块，可以选择要生成的报表类型和期间，系统会自动汇总数据并生成报表。报表生成后可以进行审核和打印。
