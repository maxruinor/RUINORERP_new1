# 库存管理模块

## 模块概述

库存管理模块是企业ERP系统的核心基础模块，负责管理企业的所有存货资产。该模块不仅服务于采购和销售业务，还为生产、财务、报表等多个模块提供数据支持。库存管理的核心目标是确保账实相符，即系统中的库存数量与实际库存保持一致。

库存管理模块支持多仓库、多库位的管理，能够跟踪产品的批次、序列号等详细信息。系统提供完善的库存预警机制，帮助企业避免缺货或积压风险。同时，模块支持多种库存业务操作，包括入库、出库、盘点、调拨、报损等，满足企业日常库存管理的各种需求。

## 核心业务流程

### 1. 入库业务流程

```
采购入库 → 入库检验 → 确认入库 → 更新库存 → 生成应付账款
```

**流程说明**：

1. 采购入库：采购订单到货后，创建采购入库单
2. 入库检验：对到货商品进行质量检验
3. 确认入库：检验合格后确认入库，更新库存
4. 更新库存：系统自动更新库存数量和成本
5. 生成应付账款：入库后生成对应的应付账款

### 2. 出库业务流程

```
销售出库 → 库存检查 → 确认出库 → 更新库存 → 生成应收账款
```

**流程说明**：

1. 销售出库：根据销售订单创建销售出库单
2. 库存检查：系统检查库存是否充足
3. 确认出库：确认出库后扣减库存
4. 更新库存：系统自动更新库存数量
5. 生成应收账款：出库后生成对应的应收账款

### 3. 盘点业务流程

```
盘点计划 → 盘点任务 → 实地盘点 → 差异录入 → 差异审核 → 账务调整
```

**流程说明**：

1. 盘点计划：制定盘点范围、时间和人员安排
2. 盘点任务：向相关人员下达盘点任务
3. 实地盘点：到仓库实地清点存货数量
4. 差异录入：将盘点数量录入系统
5. 差异审核：审核盘点差异，确认调整方案
6. 账务调整：根据审核结果调整库存账面

### 4. 调拨业务流程

```
调拨申请 → 调拨审核 → 调出发货 → 调入收货 → 完成调拨
```

**流程说明**：

1. 调拨申请：填写调拨申请，注明调出仓库、调入仓库和商品
2. 调拨审核：审批调拨申请
3. 调出发货：从调出仓库发货，扣减库存
4. 调入收货：调入仓库确认收货，增加库存
5. 完成调拨：调拨单状态更新为完成

## 关键业务规则

### 仓库管理

1. **仓库定义**：系统支持多个仓库，每个仓库可以有多个库位
2. **仓库类型**：普通仓库、虚拟仓库、委外仓库等
3. **仓库权限**：可以设置用户对仓库的操作权限
4. **仓库容量**：可以设置仓库的最大容量限制

### 库存数量

1. **可用库存**：可供销售的库存数量
2. **占用库存**：已占用但未出库的库存
3. **冻结库存**：被冻结不能使用的库存
4. **在途库存**：已发货但未收货的库存

### 成本计算

1. **成本方法**：支持先进先出、后进先出、加权平均等多种成本计算方法
2. **成本更新**：入库和出库时自动更新库存成本
3. **成本调整**：支持成本调整单进行调整
4. **成本查询**：可以查询任意时点的库存成本

### 批次管理

1. **批次号**：系统自动生成或手工录入批次号
2. **批次跟踪**：跟踪批次的入库、出库和库存情况
3. **批次有效期**：记录批次的生产日期和有效期
4. **先进先出**：出库时按批次优先使用先入库的货物

### 序列号管理

1. **序列号录入**：入库时录入或自动生成序列号
2. **序列号跟踪**：跟踪每个序列号的所有出入库记录
3. **序列号查询**：可以查询序列号的当前状态和历史
4. **序列号使用**：出库时需要选择具体的序列号

## 常见业务场景

### 场景1：库存查询

**适用情况**：查看当前库存情况

**操作步骤**：
1. 进入库存查询界面
2. 可以按仓库、产品、库位等条件查询
3. 查看各仓库的库存数量、可用数量、占用数量
4. 查看库存的成本和金额

### 场景2：库存盘点

**适用情况**：定期或不定期盘点库存，确保账实相符

**操作步骤**：
1. 创建盘点单，选择盘点范围
2. 系统生成盘点明细
3. 实地盘点，录入实际数量
4. 系统计算差异
5. 审核差异，调整库存

### 场景3：库存调拨

**适用情况**：将产品从一个仓库调拨到另一个仓库

**操作步骤**：
1. 创建调拨单
2. 选择调出仓库和调入仓库
3. 选择要调拨的产品和数量
4. 审核后执行调拨
5. 调出仓库出库，调入仓库入库

### 场景4：库存报损

**适用情况**：库存商品发生损坏或过期，需要报损处理

**操作步骤**：
1. 创建报损单
2. 选择报损仓库和产品
3. 填写报损数量和原因
4. 审核后扣减库存
5. 计入损失成本

### 场景5：库存预警

**适用情况**：设置库存预警，及时发现库存异常

**操作步骤**：
1. 进入库存预警设置
2. 为产品设置安全库存、最低库存、最高库存
3. 系统自动监控库存变化
4. 当库存低于最低或高于最高时提醒
5. 根据提醒进行补货或调整

## 数据统计与报表

### 关键指标

- **库存周转率**：销售成本/平均库存，反映库存流动速度
- **库存周转天数**：365/周转率，反映库存平均持有天数
- **库存准确率**：盘点准确数量/盘点总数，反映账实相符程度
- **缺货率**：缺货次数/订货次数，反映供应保障能力

### 常用报表

1. **库存台账**：记录每种产品的出入库明细和当前余额
2. **库龄分析**：按产品存放时间分析库存结构
3. **库存周转分析**：分析库存周转情况和趋势
4. **安全库存分析**：评估安全库存设置是否合理
5. **库存ABC分析**：按价值和数量对库存进行分类管理
6. **积压分析**：识别长期积压的库存商品

## 常见问题

### Q1：库存数量与实际不符怎么办？

A：首先需要进行库存盘点，找出差异原因。然后根据差异原因进行处理：如果是录入错误，修正录入数据；如果是管理问题，加强出入库管理；如果是损耗，进行报损处理。

### Q2：如何查看某产品的历史出入库记录？

A：在库存台账中可以查询任意产品的所有出入库明细，包括时间、数量、单价、金额、关联单据等信息。

### Q3：库存成本如何计算？

A：系统支持多种成本计算方法（先进先出、加权平均等），可以在系统参数中设置。成本会在每次入库和出库时自动更新。

### Q4：如何设置安全库存？

A：在库存预警设置中，可以为每个产品设置安全库存、最低库存和最高库存。系统会根据设置自动进行库存预警。

### Q5：调拨完成后库存如何变化？

A：调拨单审核后，调出仓库的库存会减少，调入仓库的库存会增加。两个仓库的库存台账都会记录这笔调拨业务。
