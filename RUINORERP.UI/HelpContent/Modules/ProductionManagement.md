# 生产管理模块

## 模块概述

生产管理模块是企业ERP系统的核心模块，负责处理从生产计划、生产加工到成品入库的完整生产业务流程。该模块与销售管理、采购管理、库存管理、财务管理等模块紧密集成，能够根据销售订单或库存需求自动生成生产计划，并跟踪整个生产过程。

生产管理模块主要包括生产计划、生产订单、BOM管理、车间作业、质量检验、成本核算等功能。通过与MRP（物料需求计划）的集成，系统能够根据销售订单自动计算所需的物料需求，生成采购建议和生产计划，确保生产的顺利进行。

## 核心业务流程

### 1. 标准生产流程

```
销售订单 → MRP运算 → 生产计划 → 生产订单 → 生产领料 → 生产入库 → 成品库存
```

**流程说明**：

1. 销售订单：销售部门创建销售订单
2. MRP运算：系统根据销售订单运算物料需求
3. 生产计划：生成生产计划单，安排生产任务
4. 生产订单：创建生产订单，下达生产任务
5. 生产领料：根据BOM分解，领取生产所需物料
6. 生产入库：完成生产后，成品入库
7. 成品库存：产品入库后进入成品库存

### 2. 库存补货流程

```
安全库存预警 → MRP运算 → 生产计划 → 生产订单 → 生产执行 → 入库
```

**流程说明**：

1. 安全库存预警：库存低于安全库存时触发
2. MRP运算：系统运算补货需求
3. 生成生产计划：自动创建生产计划
4. 下达生产订单：生成生产任务
5. 执行生产：领料、生产、入库

### 3. 外协加工流程

```
生产订单 → 外协发料 → 外协加工 → 外协收货 → 成品入库 → 结算加工费
```

**特殊处理**：

- 需要进行发料操作，将物料发给外协厂商
- 加工完成后需要收货入库
- 加工费用需要单独核算和结算
- 需要跟踪外协加工进度

## 关键业务规则

### 生产计划管理

1. **计划来源**：可从销售订单、MRP建议、手工创建
2. **计划排程**：根据产能和交期自动排产
3. **计划变更**：可以调整计划数量和日期
4. **计划跟踪**：跟踪计划的执行进度

### 物料控制

1. **BOM（物料清单）**：定义产品的物料组成
2. **物料需求**：根据BOM和数量计算所需物料
3. **物料检查**：检查库存是否满足需求
4. **缺料提醒**：缺料时自动提醒采购

### 生产执行

1. **生产派工**：将生产任务分配到车间/班组
2. **进度汇报**：汇报生产进度和完成情况
3. **质量检验**：生产过程中和完成后进行检验
4. **入库确认**：完成后确认入库

### 成本控制

1. **材料成本**：按实际领料计算材料成本
2. **人工成本**：记录工时和人工费用
3. **制造费用**：分摊制造费用
4. **成本分析**：对比实际成本与标准成本

## 常见业务场景

### 场景1：按订单生产

**适用情况**：根据销售订单进行生产

**操作步骤**：
1. 销售订单确认后
2. MRP运算物料需求
3. 生成生产计划
4. 创建生产订单
5. 领料生产
6. 成品入库
7. 销售出库

### 场景2：库存生产

**适用情况**：根据安全库存进行生产

**操作步骤**：
1. 库存低于安全库存
2. MRP运算补货需求
3. 生成生产计划
4. 创建生产订单
5. 生产入库
6. 补充库存

### 场景3：批量生产

**适用情况**：大批量生产同一产品

**操作步骤**：
1. 创建生产订单，设置大批量
2. 分批领料，分批入库
3. 跟踪累计完成情况
4. 全部完成后关闭订单

## 数据统计与报表

### 关键指标

- **生产完成率**：已完成订单数/计划订单数
- **交期达成率**：按时完成订单数/总订单数
- **产品合格率**：合格产品数/生产总数
- **单位成本**：总成本/产量

### 常用报表

1. **生产进度报表**：跟踪生产订单执行进度
2. **在制明细表**：显示当前在制产品数量
3. **完工汇总表**：统计已完成的生产情况
4. **成本分析表**：分析产品成本构成
5. **物料消耗表**：统计材料消耗情况

## 系统集成

### 与销售系统集成

- 销售订单触发生产计划
- 成品用于销售出库
- 销售预测影响生产计划

### 与采购系统集成

- 物料需求生成采购建议
- 采购物料用于生产
- 采购成本影响产品成本

### 与库存系统集成

- 原材料出库用于生产
- 成品入库增加库存
- 在制品库存跟踪

### 与财务系统集成

- 生产成本自动核算
- 人工费用计入成本
- 成品成本更新

## 常见问题

### Q1：如何查看生产进度？

A：在生产订单列表中可以查看每笔订单的执行进度，包括领料、入库、检验等状态。

### Q2：缺料时怎么办？

A：系统会提示缺料信息，可以根据提示生成采购建议，或调整生产计划。

### Q3：BOM如何设置？

A：在BOM管理中为产品定义物料清单，包括主料和辅料、用量标准等。

### Q4：生产成本如何计算？

A：系统根据实际领料、工时和制造费用自动归集和计算产品成本。

### Q5：如何处理生产异常？

A：可以记录生产异常情况，进行返工或报废处理，并分析原因改进。
