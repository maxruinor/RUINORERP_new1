# 系统设置模块

## 模块概述

系统设置模块是整个ERP系统的基础配置中心，用于管理系统运行所需的各种参数和规则。合理的系统设置是系统正常运行的基础，也是满足企业个性化需求的重要手段。

系统设置与所有业务模块紧密关联。系统参数影响所有业务功能的运行方式，权限设置控制用户对各功能的使用，编码规则定义单据编号的生成方式。系统管理员需要根据企业的实际需求和业务流程，配置合适的系统参数。

## 核心功能

### 1. 系统参数设置

```
基本参数 → 业务参数 → 财务参数 → 生产参数
```

**参数类型**：

- **基本参数**：公司信息、账套设置、语言时区
- **业务参数**：库存控制、信用控制、审批流程
- **财务参数**：成本方法、凭证设置、税率
- **生产参数**：MRP参数、工艺参数

### 2. 编码规则管理

```
编码规则 → 单据编号 → 自动编号
```

**功能说明**：

- 定义各类单据的编号规则
- 支持日期、流水号、自定义前缀
- 可以按年份、月份重置

### 3. 权限管理

```
角色管理 → 用户管理 → 功能权限 → 数据权限
```

**功能说明**：

- 定义系统角色
- 分配用户到角色
- 设置角色功能权限
- 配置数据权限范围

### 4. 个性化设置

```
界面设置 → 快捷键设置 → 默认值设置
```

**功能说明**：

- 调整界面显示
- 自定义快捷键
- 设置常用默认值

## 常用设置说明

### 系统基本参数

| 参数 | 说明 | 建议值 |
|------|------|--------|
| 公司名称 | 系统显示的公司名 | 实际公司名称 |
| 会计期间 | 财务核算期间 | 按自然月或业务周期 |
| 启用负库存 | 是否允许负库存 | 一般不允许 |
| 小数位数 | 数量和金额精度 | 数量2位，金额4位 |

### 业务控制参数

| 参数 | 说明 | 说明 |
|------|------|------|
| 信用控制 | 超信用是否阻止 | 建议启用 |
| 库存预警 | 低于安全库存提醒 | 建议启用 |
| 订单审核 | 订单是否需要审核 | 根据需要 |
| 批次管理 | 是否启用批次管理 | 根据产品特性 |

### 编码规则示例

| 单据类型 | 编码规则 | 示例 |
|----------|----------|------|
| 销售订单 | SO + 年月 + 4位流水 | SO2024010001 |
| 采购订单 | PO + 年月 + 4位流水 | PO2024010001 |
| 销售出库 | SOUT + 年月 + 4位流水 | SOUT2024010001 |
| 产品资料 | P + 分类 + 3位流水 | P001001 |

## 常见设置场景

### 场景1：设置编码规则

**目标**：定义单据编号的生成方式

**操作步骤**：
1. 进入编码规则设置
2. 选择要设置的单据类型
3. 定义编码格式（前缀+日期+流水号）
4. 设置流水号规则（位数、重置周期）
5. 保存设置

### 场景2：配置用户权限

**目标**：为用户分配功能使用权限

**操作步骤**：
1. 创建或编辑角色
2. 勾选角色可使用的功能
3. 保存角色
4. 将用户分配到对应角色
5. 用户登录后获得相应权限

### 场景3：设置业务控制参数

**目标**：配置业务控制规则

**操作步骤**：
1. 进入业务参数设置
2. 找到要修改的参数项
3. 设置参数值
4. 保存设置
5. 参数立即生效

## 权限管理

### 角色类型

| 角色 | 说明 | 权限范围 |
|------|------|----------|
| 系统管理员 | 系统最高权限 | 所有功能 |
| 业务管理员 | 业务管理权限 | 业务模块管理 |
| 普通用户 | 一般使用权限 | 业务操作 |
| 只读用户 | 仅查看权限 | 查询功能 |

### 权限设置原则

1. **最小权限**：只分配必要的权限
2. **职责分离**：关键操作需要多人协作
3. **定期审查**：定期检查权限分配

## 常见问题

### Q1：修改参数后需要重启吗？

A：大部分参数修改后立即生效，无需重启。少部分核心参数可能需要。

### Q2：编码规则可以修改吗？

A：可以修改，但只对新增单据生效，已生成的单据编号不变。

### Q3：如何添加新用户？

A：在用户管理中添加用户，分配角色和所属组织。

### Q4：忘记管理员密码怎么办？

A：需要通过数据库或系统维护工具重置。

### Q5：如何设置数据权限？

A：可以按组织、部门或自定义范围设置数据查看权限。

## 快捷键

| 快捷键 | 功能 |
|-------|------|
| F1 | 显示帮助 |
| F5 | 刷新数据 |
| Ctrl+S | 保存设置 |

## 注意事项

1. **谨慎修改**：系统参数修改可能影响业务运行
2. **记录变更**：重要参数修改建议记录备查
3. **测试验证**：生产环境修改前建议在测试环境验证
