# RUINOR ERP 帮助系统

欢迎使用RUINOR ERP智能帮助系统!

## 快速开始

### 如何使用帮助系统

1. **F1键帮助**: 在任何界面按F1键,查看当前焦点控件的帮助信息
2. **智能提示**: 将鼠标悬停在控件上,会自动显示简短的帮助提示
3. **帮助按钮**: 帮助图标按钮点击查看详细帮助
4. **搜索帮助**: 使用搜索功能查找相关帮助内容

### 帮助级别

本帮助系统提供四个级别的帮助:

- **模块级**: 业务模块的整体介绍和功能概述
- **窗体级**: 单个窗体的功能说明和操作流程
- **控件级**: 控件的使用方法和注意事项
- **字段级**: 字段的详细说明和输入要求
