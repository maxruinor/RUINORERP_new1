# 销售管理模块

## 模块概述
销售管理模块是RUINORERP系统的核心模块之一，负责处理公司所有销售业务流程，从客户管理、销售订单创建、出库发货到应收账款管理，提供完整的销售业务解决方案。

## 主要功能

### 1. 客户管理
- 客户信息维护
- 客户分类管理
- 客户信用额度控制
- 客户交易记录查询

### 2. 销售订单
- 销售订单创建与编辑
- 订单审批流程
- 订单状态跟踪
- 订单变更与取消

### 3. 出库管理
- 销售出库单生成
- 库存自动扣减
- 出库单打印
- 物流信息管理

### 4. 收款管理
- 应收账款查询
- 收款单录入
- 收款核销
- 账龄分析

### 5. 报表分析
- 销售统计报表
- 客户销售分析
- 产品销售排行
- 销售趋势分析

## 业务流程

```
客户信息 → 销售报价 → 销售订单 → 出库发货 → 收款核销
```

## 使用权限

| 角色 | 客户管理 | 销售订单 | 出库管理 | 收款管理 | 报表查看 |
|------|---------|---------|---------|---------|---------|
| 销售经理 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 销售员 | ✓ | ✓ | ✗ | ✗ | ✓ |
| 仓管员 | ✗ | ✓ | ✓ | ✗ | ✓ |
| 财务 | ✗ | ✗ | ✗ | ✓ | ✓ |

## 注意事项
1. 所有销售订单必须关联有效客户
2. 出库前需确认库存充足
3. 超额出库需要特殊审批
4. 收款核销需匹配对应的销售订单
