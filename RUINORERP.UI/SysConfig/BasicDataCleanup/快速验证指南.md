## 🔍 常见问题排查

### 问题1: 仍然出现外键约束错误

**症状**：
```
DELETE 语句与 REFERENCE 约束"FK_XXX"冲突
```

**排查步骤**：
1. 查看日志中的详细错误信息
2. 检查是否提到了具体的表和字段
3. 确认 `tb_ProdCategoriesController.BaseDeleteByNavAsync` 方法中是否包含了所有导航属性的 Include

**解决方案**：
```csharp
// 检查 Controller 中的实现
public override async Task<bool> BaseDeleteByNavAsync(tb_ProdCategories entity)
{
    bool rs = await _db.DeleteNav<tb_ProdCategories>(m => m.Category_ID == entity.Category_ID)
        .Include(m => m.tb_Prods)  // ← 确保包含这个
        .ExecuteCommandAsync();
    return rs;
}
```

### 问题2: 测试模式统计不准确

**症状**：
- 预览显示删除 1 条，但实际删除了 100 条

**排查步骤**：
1. 检查日志中是否有 `[测试]` 开头的统计信息
2. 确认 `SimulateDeepLevelDeleteAsync` 方法是否被调用
3. 检查实体的导航属性是否正确配置

**解决方案**：
```csharp
// 检查 Model 中的导航属性配置
public class tb_ProdCategories : BaseEntity
{
    [SugarColumn(IsPrimaryKey = true)]
    public long Category_ID { get; set; }
    
    // 确保有这个导航属性
    [Navigate(NavigateType.OneToMany, nameof(tb_Prod.Category_ID))]
    public List<tb_Prod> tb_Prods { get; set; }
}
```

### 问题3: 删除速度过慢

**症状**：
- 删除少量数据耗时超过 10 秒

**优化建议**：
1. 检查数据库中是否有大量关联数据
2. 考虑分批删除（每次 50-100 条）
3. 检查数据库索引是否优化
