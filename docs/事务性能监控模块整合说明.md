## 支持类定义

### TransactionDuration
记录每次事务执行的详细信息：
```csharp
public class TransactionDuration
{
    public string Operation { get; set; }      // commit/rollback
    public string Caller { get; set; }         // 调用者方法
    public double DurationSeconds { get; set; }// 持续时间
    public bool Succeeded { get; set; }        // 是否成功
    public string TableName { get; set; }      // 表名
    public DateTime Timestamp { get; set; }    // 时间戳
}
```

### DeadlockStatistic
记录死锁事件的详细信息：
```csharp
public class DeadlockStatistic
{
    public string TableName { get; set; }   // 死锁涉及的表
    public string Operation { get; set; }   // 操作类型
    public TimeSpan Duration { get; set; }  // 等待时间
    public string Sql { get; set; }         // SQL 语句
    public string Victim { get; set; }      // 牺牲者
    public DateTime Time { get; set; }      // 发生时间
}
```

### TableStatistics
统计表级别的性能指标：
```csharp
public class TableStatistics
{
    public string TableName { get; set; }       // 表名
    public long TransactionCount { get; set; }  // 事务次数
    public double TotalDuration { get; set; }   // 总耗时
    public double AverageDuration =>            // 平均耗时
        TransactionCount > 0 ? TotalDuration / TransactionCount : 0;
}
```

### 2. 分析慢事务
```csharp
var stats = TransactionMetrics.GetStatistics();
var slowTransactions = stats.TopLongTransactions
    .Where(t => t.DurationSeconds > 1.0)
    .ToList();

foreach (var tx in slowTransactions)
{
    Console.WriteLine($"慢事务：{tx.Caller}, 耗时：{tx.DurationSeconds:F2}秒");
}
```

### 3. 识别热点表
```csharp
var hotTables = stats.HotTables
    .Where(t => t.TransactionCount > 1000)
    .OrderByDescending(t => t.AverageDuration)
    .ToList();

foreach (var table in hotTables)
{
    Console.WriteLine($"热点表：{table.TableName}, " +
                     $"次数：{table.TransactionCount}, " +
                     $"平均：{table.AverageDuration:F3}秒");
}
```

### 4. 分析死锁模式
```csharp
var deadlocks = stats.RecentDeadlocks
    .GroupBy(d => d.TableName)
    .Select(g => new 
    {
        TableName = g.Key,
        Count = g.Count(),
        AvgDuration = g.Average(d => d.Duration.TotalMilliseconds)
    })
    .OrderByDescending(x => x.Count)
    .ToList();

foreach (var dl in deadlocks)
{
    Console.WriteLine($"表：{dl.TableName}, " +
                     $"死锁次数：{dl.Count}, " +
                     $"平均耗时：{dl.AvgDuration:F0}ms");
}
```

## 性能优化

### 线程安全
- ✅ 使用 `ConcurrentDictionary`、`ConcurrentQueue`、`ConcurrentBag`
- ✅ 细粒度锁（`_durationLock`、`_deadlockLock`、`_topTransactionsLock`）
- ✅ 避免大锁竞争

### 内存管理
- ✅ 限制最大记录数（MAX_DURATION_RECORDS = 10000）
- ✅ 定期清理旧死锁记录（DEADLOCK_RETENTION_DAYS = 7）
- ✅ 限制 Top N 数量（TOP_N_COUNT = 20）
