### 3.3 简化 GetPrimaryKeyFieldName()

**当前实现**（4层降级）：
```csharp
private string GetPrimaryKeyFieldName(Type entityType) {
    // 1. 查 EntityRegistry
    // 2. 查 BaseEntity.GetPrimaryKeyColName()
    // 3. 反射查 SugarColumn 特性
    // 4. 默认返回 "ID"
}
```

**优化后**（1层查询）：
```csharp
private string GetPrimaryKeyFieldName(Type entityType) {
    var metadata = EntityRegistry.GetByType(entityType);
    if (metadata == null || string.IsNullOrEmpty(metadata.PrimaryKeyName)) {
        throw new InvalidOperationException(
            $"实体 {entityType.Name} 未在 EntityRegistry 中注册或缺少主键配置");
    }
    return metadata.PrimaryKeyName;
}
```

**前提条件**：确保所有导入涉及的实体都在 EntityRegistry 中注册

```csharp
// XML 转 JSON 转换工具
public class ConfigMigrationTool
{
    public static void ConvertXmlToJson(string xmlPath, string jsonPath)
    {
        var xmlSerializer = new XmlSerializer(typeof(ImportConfiguration));
        using (var reader = new StreamReader(xmlPath))
        {
            var config = (ImportConfiguration)xmlSerializer.Deserialize(reader);
            
            // 转换为 ImportProfile (JSON格式)
            var profile = ConvertToImportProfile(config);
            
            var json = JsonConvert.SerializeObject(profile, Formatting.Indented);
            File.WriteAllText(jsonPath, json);
        }
    }
}
```
