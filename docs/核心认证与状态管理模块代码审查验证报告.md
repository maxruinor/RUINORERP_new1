private async Task<int> GetMaxConcurrentUsersAsync()

    lock (_cacheLock)

            return _cachedMaxConcurrentUsers.Value;
    }

### 3. P1-4: GlobalStateRulesManager.cs 线程安全问题

**文件位置**: [GlobalStateRulesManager.cs#L83-96](file:///e:/CodeRepository/SynologyDrive/RUINORERP/RUINORERP.Model/Base/StatusManager/GlobalStateRulesManager.cs#L83-L96)

**问题代码**:
```csharp
public void SetSubmitModifyRuleMode(SubmitModifyRuleMode mode)
{
    if (submitModifyRuleMode != mode)
    {
        submitModifyRuleMode = mode;
        
        if (_isInitialized)
        {
            ResetAllRules();      // ⚠️ 非原子操作
            InitializeAllRules(); // 中间状态可能被其他线程访问
        }
    }
}
```

**问题分析**:
- `ResetAllRules()`和`InitializeAllRules()`之间不是原子操作
- 其他线程可能在规则重置后、初始化前访问规则字典
- `_isInitialized`标志未使用锁保护
- 整个类中都不存在 `_initLock` 字段

**修复建议**:
```csharp
private readonly object _initLock = new object();

public void SetSubmitModifyRuleMode(SubmitModifyRuleMode mode)
{
    lock (_initLock)
    {
        if (submitModifyRuleMode != mode)
        {
            submitModifyRuleMode = mode;
            if (_isInitialized)
            {
                ResetAllRules();
                InitializeAllRules();
            }
        }
    }
}
```

**验证结论**: 问题确实存在，优化方案合理
